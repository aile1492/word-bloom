## 1. 온보딩 전략 개요

### 1.1 설계 원칙

| 원칙 | 설명 |
|------|------|
| **Learn by Doing** | 텍스트 설명 최소화, 실제 퍼즐 플레이를 통해 학습 |
| **점진적 복잡도** | Stage 1(기본) → Stage 2(확장) → Stage 3(자유) |
| **최소 텍스트** | 가이드 메시지는 1~2문장 이내, 아이콘/애니메이션 활용 |
| **스킵 불가** | 첫 3 스테이지는 필수 (핵심 메커닉 학습 보장) |
| **부드러운 전환** | 튜토리얼에서 일반 게임플레이로 자연스럽게 전환 |

### 1.2 튜토리얼 구간 개요

```
첫 실행
  │
  ▼
[언어 선택] → [닉네임 + 아바타 (선택)] → [Stage 1: 기본 드래그]
                                              │
                                              ▼
                                        [Stage 2: 전방향]
                                              │
                                              ▼
                                        [Stage 3: 자유 플레이]
                                              │
                                              ▼
                                        tutorial_completed = true
                                              │
                                              ▼
                                        [일반 게임 진행 (Stage 4~)]
```

### 1.3 튜토리얼 vs 일반 게임 차이점

| 항목 | 튜토리얼 (Stage 1-3) | 일반 게임 (Stage 4+) |
|------|---------------------|---------------------|
| 가이드 오버레이 | 표시 (단계적 감소) | 미표시 |
| 힌트 버튼 | 숨김 | 표시 |
| 타이머 | 없음 | 모드에 따라 |
| 일시정지 | 불가 | 가능 |
| 격자 크기 | 고정 (3x3, 4x4, 5x5) | GridCalculator 계산 |
| 단어 수 | 고정 (2, 3, 4) | GridCalculator 계산 |
| 단어 배치 방향 | 제한적 (단계적 확장) | 8방향 전체 |
| 코인 획득 | 없음 (또는 최소) | 정상 획득 |
| 결과 화면 | 간소화 | 전체 표시 |

## 2. 첫 실행 플로우

### 2.1 전체 플로우 다이어그램

```
[앱 최초 실행]
     │
     ▼
SaveManager.load()
     │
     ├── tutorial_completed == true → [일반 TitleScreen]
     │
     └── tutorial_completed == false (첫 실행)
           │
           ▼
     ┌──────────────────────┐
     │  언어 선택 화면       │
     │  [English] [한국어]   │
     └──────────┬───────────┘
                │
                ▼
     ┌──────────────────────────────────┐
     │  닉네임 + 아바타 설정 (선택적)    │
     │  [닉네임 입력]                    │
     │  [아바타 선택]                    │
     │  [건너뛰기] [확인]               │
     └──────────┬──────────┬────────────┘
                │          │
           (확인)     (건너뛰기)
                │          │
                ▼          ▼
     ┌──────────────────────┐
     │  Tutorial Stage 1     │
     │  (자동 시작)           │
     └──────────────────────┘
```

### 2.2 언어 선택 화면

```gdscript
## language_select_screen.gd
## 첫 실행 시 표시되는 언어 선택 화면.
extends Control

signal language_selected(language_index: int)

@onready var english_button: Button = $VBoxContainer/EnglishButton
@onready var korean_button: Button = $VBoxContainer/KoreanButton


func _ready() -> void:
	english_button.pressed.connect(_on_english_pressed)
	korean_button.pressed.connect(_on_korean_pressed)


func _on_english_pressed() -> void:
	SaveManager.update_setting("language_index", 0)
	language_selected.emit(0)


func _on_korean_pressed() -> void:
	SaveManager.update_setting("language_index", 1)
	language_selected.emit(1)
```

### 2.3 닉네임 + 아바타 설정

닉네임과 아바타 설정은 **선택적**이다. 건너뛰기를 선택하면 기본 닉네임(`Guest_xxxxx`)과 기본 아바타(index 0)를 사용한다.

```gdscript
## profile_setup_screen.gd
## 첫 실행 시 닉네임과 아바타를 설정하는 화면.
extends Control

signal setup_completed()
signal setup_skipped()

@onready var nickname_input: LineEdit = $VBoxContainer/NicknameInput
@onready var avatar_grid: GridContainer = $VBoxContainer/AvatarGrid
@onready var confirm_button: Button = $VBoxContainer/ConfirmButton
@onready var skip_button: Button = $VBoxContainer/SkipButton

var _selected_avatar: int = 0


func _ready() -> void:
	confirm_button.pressed.connect(_on_confirm)
	skip_button.pressed.connect(_on_skip)
	nickname_input.text = SaveManager.get_nickname()


func _on_confirm() -> void:
	var nickname: String = nickname_input.text.strip_edges()
	if nickname.length() > 0:
		SaveManager.set_nickname(nickname)
	SaveManager.set_avatar_index(_selected_avatar)
	SaveManager.save()
	setup_completed.emit()


func _on_skip() -> void:
	# 기본 닉네임과 아바타 유지
	setup_skipped.emit()
```

### 2.4 첫 실행 관리자

```gdscript
## first_run_manager.gd
## 첫 실행 온보딩 시퀀스를 관리한다.
## ScreenManager와 연동하여 화면을 순차적으로 표시한다.
class_name FirstRunManager
extends RefCounted


## 첫 실행 여부를 확인하고 온보딩 시퀀스를 시작한다.
## true를 반환하면 첫 실행이므로 온보딩이 시작된 것이다.
static func check_and_start() -> bool:
	if SaveManager.is_tutorial_completed():
		return false

	# 첫 실행: 온보딩 시작
	_start_onboarding_sequence()
	return true


## 온보딩 시퀀스를 시작한다.
static func _start_onboarding_sequence() -> void:
	# 1. 언어 선택 Push
	ScreenManager.push_screen("language_select", {
		"on_complete": func(lang_index: int) -> void:
			# 2. 프로필 설정 Push
			ScreenManager.push_screen("profile_setup", {
				"on_complete": func() -> void:
					# 3. 튜토리얼 Stage 1 시작
					TutorialManager.start_tutorial()
			})
	})
```

## 3. 인터랙티브 튜토리얼 (Stage 1-3)

### 3.1 Stage 1: 기본 드래그 (Basic Drag)

가장 간단한 환경에서 드래그 인터랙션의 기본을 학습한다.

| 항목 | 설정 |
|------|------|
| **격자 크기** | 3x3 |
| **단어 수** | 2개 |
| **배치 방향** | 수평(Horizontal)만 |
| **가이드 수준** | 최대 (애니메이션 손가락 + 하이라이트 + 툴팁) |

**고정 퍼즐 데이터 (EN):**

```
C A T
D O G
X Y Z
```
- 단어 1: "CAT" (0,0) → (1,0) Right
- 단어 2: "DOG" (0,1) → (1,0) Right

**고정 퍼즐 데이터 (KO):**

```
호 랑 이
강 아 지
나 비 꽃
```
- 단어 1: "호랑이" (0,0) → (1,0) Right
- 단어 2: "강아지" (0,1) → (1,0) Right

**가이드 시퀀스:**

```
Step 1: 화면 전체 딤 + 격자 스포트라이트
        툴팁: "격자 안에 숨겨진 단어를 찾아보세요!"
        [알겠어요!] 버튼

Step 2: 단어 목록(WordBank) 스포트라이트
        툴팁: "이 단어들을 찾아야 해요"
        (자동 진행, 2초 후)

Step 3: 첫 번째 단어 하이라이트 (셀 반짝임)
        애니메이션 손가락이 C → A → T를 드래그하는 동작 시연
        툴팁: "손가락을 드래그해서 단어를 찾아보세요!"

Step 4: 플레이어 입력 대기
        첫 번째 단어만 드래그 가능 (나머지 셀 비활성화)
        성공 시 → 축하 이펙트 + "잘했어요!"

Step 5: 두 번째 단어 하이라이트 (약간의 힌트만)
        애니메이션 손가락 없이 첫 글자만 반짝임
        플레이어가 직접 드래그

Step 6: 모든 단어 발견 → 간소화된 결과 화면
        "훌륭해요! 다음 스테이지로!"
        (Next 버튼만 표시, Retry/Home 숨김)
```

**Stage 1 튜토리얼 퍼즐 설정 코드:**

```gdscript
## tutorial_puzzles.gd
## 튜토리얼 전용 고정 퍼즐 데이터를 제공한다.
class_name TutorialPuzzles
extends RefCounted


## Stage 1용 고정 퍼즐을 반환한다.
static func get_stage_1_puzzle(language_index: int) -> GridData:
	var data := GridData.new()
	data.stage = 1
	data.width = 3
	data.height = 3

	if language_index == 0:  # English
		data.language = "en"
		data.grid = [
			["C", "A", "T"],
			["D", "O", "G"],
			["X", "Y", "Z"],
		]
		data.placed_words = [
			PlacedWord.create("cat", "CAT", Vector2i(0, 0), Vector2i(1, 0)),
			PlacedWord.create("dog", "DOG", Vector2i(0, 1), Vector2i(1, 0)),
		]
	else:  # Korean
		data.language = "ko"
		data.grid = [
			["호", "랑", "이"],
			["강", "아", "지"],
			["나", "비", "꽃"],
		]
		data.placed_words = [
			PlacedWord.create("호랑이", "호랑이", Vector2i(0, 0), Vector2i(1, 0)),
			PlacedWord.create("강아지", "강아지", Vector2i(0, 1), Vector2i(1, 0)),
		]

	# 색상 인덱스 할당
	for i in range(data.placed_words.size()):
		data.placed_words[i].color_index = i

	return data
```

### 3.2 Stage 2: 수직 + 대각선 (Vertical + Diagonal)

수평 외의 방향을 학습한다.

| 항목 | 설정 |
|------|------|
| **격자 크기** | 4x4 |
| **단어 수** | 3개 |
| **배치 방향** | 수평 + 수직 + 대각선 (전 방향) |
| **가이드 수준** | 중간 (간단한 툴팁, 애니메이션 손가락 없음) |

**가이드 시퀀스:**

```
Step 1: 간단한 툴팁 표시
        "단어는 모든 방향으로 숨겨져 있어요!"
        방향 아이콘 애니메이션: → ↓ ↘ ↙
        [알겠어요!] 버튼

Step 2: 플레이어 자유 플레이
        첫 번째 단어 발견 시 → "좋아요! 계속 찾아보세요!"
        (이후 추가 가이드 없음)

Step 3: 모든 단어 발견 → 간소화된 결과 화면
```

**Stage 2 퍼즐 설정:**

```gdscript
## Stage 2용 고정 퍼즐을 반환한다.
static func get_stage_2_puzzle(language_index: int) -> GridData:
	var data := GridData.new()
	data.stage = 2
	data.width = 4
	data.height = 4

	if language_index == 0:  # English
		data.language = "en"
		data.grid = [
			["S", "U", "N", "A"],
			["T", "R", "E", "E"],
			["A", "B", "I", "D"],
			["R", "C", "D", "K"],
		]
		data.placed_words = [
			PlacedWord.create("sun", "SUN", Vector2i(0, 0), Vector2i(1, 0)),    # 수평 →
			PlacedWord.create("star", "STAR", Vector2i(0, 0), Vector2i(0, 1)),   # 수직 ↓
			PlacedWord.create("tree", "TREE", Vector2i(0, 1), Vector2i(1, 0)),   # 수평 →
		]
	else:  # Korean
		data.language = "ko"
		data.grid = [
			["하", "늘", "달", "빛"],
			["바", "다", "숲", "꿈"],
			["람", "나", "무", "별"],
			["꽃", "잎", "새", "솔"],
		]
		data.placed_words = [
			PlacedWord.create("하늘", "하늘", Vector2i(0, 0), Vector2i(1, 0)),   # 수평 →
			PlacedWord.create("바다", "바다", Vector2i(0, 1), Vector2i(1, 0)),    # 수평 →
			PlacedWord.create("나무", "나무", Vector2i(1, 2), Vector2i(1, 0)),    # 수평 →
		]

	for i in range(data.placed_words.size()):
		data.placed_words[i].color_index = i

	return data
```

### 3.3 Stage 3: 자유 플레이 (Free Play)

가이드 없이 일반 게임과 동일한 환경에서 플레이한다. 이 스테이지를 클리어하면 튜토리얼이 완료된다.

| 항목 | 설정 |
|------|------|
| **격자 크기** | 5x5 (일반 Stage 1과 동일) |
| **단어 수** | 4개 |
| **배치 방향** | 8방향 전체 |
| **가이드 수준** | 없음 (일반 게임과 동일) |

```gdscript
## Stage 3은 일반 격자 생성 알고리즘을 사용한다.
## 고정 퍼즐이 아닌 랜덤 생성이지만, 튜토리얼 상태 추적은 유지한다.
static func get_stage_3_config() -> Dictionary:
	return {
		"stage": 3,
		"width": 5,
		"height": 5,
		"word_count": 4,
		"is_tutorial": true,  # 결과 화면 간소화 유지
		"use_random_generation": true,  # 랜덤 생성 사용
	}
```

### 3.4 튜토리얼 스테이지 비교 표

| 항목 | Stage 1 | Stage 2 | Stage 3 |

| 툴팁 메시지 | 3개 | 1개 | 0개 |
| 코인 지급 | 0 | 0 | 정상 |
| 결과 화면 | 간소화 | 간소화 | 간소화 |

### 4.2 guide_overlay.gd

```gdscript
## guide_overlay.gd
## 튜토리얼 가이드 오버레이를 표시한다.
## 반투명 딤, 스포트라이트, 애니메이션 손가락, 툴팁을 관리한다.
extends CanvasLayer

## --- 시그널 ---
signal step_dismissed()
signal guide_completed()

## --- 상수 ---
const DIM_ALPHA: float = 0.7
const HAND_ANIMATION_DURATION: float = 1.5  # 드래그 시연 애니메이션 시간
const TOOLTIP_APPEAR_DURATION: float = 0.3

## --- 노드 참조 ---
@onready var dim_background: ColorRect = $DimBackground
@onready var spotlight: Control = $Spotlight
@onready var animated_hand: TextureRect = $AnimatedHand
@onready var tooltip_panel: PanelContainer = $TooltipPanel
@onready var tooltip_label: Label = $TooltipPanel/VBoxContainer/MessageLabel
@onready var dismiss_button: Button = $TooltipPanel/VBoxContainer/DismissButton

## --- 상태 ---
var _current_step: int = 0
var _steps: Array[Dictionary] = []
var _hand_tween: Tween = null


func _ready() -> void:
	dismiss_button.pressed.connect(_on_dismiss_pressed)
	visible = false


## 가이드 시퀀스를 시작한다.
## steps 배열의 각 요소는 Dictionary로 다음 키를 포함한다:
##   "message": String - 툴팁 메시지
##   "spotlight_rect": Rect2 - 스포트라이트 영역 (없으면 전체 딤)
##   "hand_from": Vector2 - 손가락 시작 위치 (없으면 손가락 숨김)
##   "hand_to": Vector2 - 손가락 끝 위치
##   "auto_advance": float - 자동 진행 시간 (0이면 수동)
##   "show_button": bool - 확인 버튼 표시 여부
func start_sequence(steps: Array[Dictionary]) -> void:
	_steps = steps
	_current_step = 0
	visible = true
	_show_step(0)


## 현재 스텝을 표시한다.
func _show_step(index: int) -> void:
	if index >= _steps.size():
		_complete_sequence()
		return

	var step: Dictionary = _steps[index]

	# 딤 배경 표시
	dim_background.color = Color(0, 0, 0, DIM_ALPHA)

	# 스포트라이트 영역 설정
	if step.has("spotlight_rect"):
		spotlight.visible = true
		_set_spotlight_rect(step.spotlight_rect)
	else:
		spotlight.visible = false

	# 툴팁 메시지
	if step.has("message") and not step.message.is_empty():
		tooltip_panel.visible = true
		tooltip_label.text = step.message
		_animate_tooltip_in()
	else:
		tooltip_panel.visible = false

	# 확인 버튼
	dismiss_button.visible = step.get("show_button", true)

	# 애니메이션 손가락
	if step.has("hand_from") and step.has("hand_to"):
		animated_hand.visible = true
		_animate_hand_drag(step.hand_from, step.hand_to)
	else:
		animated_hand.visible = false
		_stop_hand_animation()

	# 자동 진행
	var auto_time: float = step.get("auto_advance", 0.0)
	if auto_time > 0.0:
		dismiss_button.visible = false
		await get_tree().create_timer(auto_time).timeout
		_advance_step()


## 손가락 드래그 애니메이션을 반복 재생한다.
func _animate_hand_drag(from_pos: Vector2, to_pos: Vector2) -> void:
	_stop_hand_animation()

	animated_hand.position = from_pos

	_hand_tween = create_tween().set_loops()
	# 시작 위치에서 잠시 대기
	_hand_tween.tween_interval(0.3)
	# 시작 시 눌림 효과 (스케일 축소)
	_hand_tween.tween_property(animated_hand, "scale", Vector2(0.8, 0.8), 0.1)
	# 드래그 이동
	_hand_tween.tween_property(animated_hand, "position", to_pos, HAND_ANIMATION_DURATION)\
		.set_ease(Tween.EASE_IN_OUT).set_trans(Tween.TRANS_SINE)
	# 뗌 효과 (스케일 복원)
	_hand_tween.tween_property(animated_hand, "scale", Vector2.ONE, 0.1)
	# 원위치 복귀 (빠르게)
	_hand_tween.tween_property(animated_hand, "position", from_pos, 0.3)
	# 잠시 대기 후 반복
	_hand_tween.tween_interval(0.5)


## 손가락 애니메이션을 중지한다.
func _stop_hand_animation() -> void:
	if _hand_tween and _hand_tween.is_running():
		_hand_tween.kill()
	_hand_tween = null


## 스포트라이트 영역을 설정한다.
## 지정된 Rect2 영역만 투명하게 표시하고 나머지는 딤 처리한다.
func _set_spotlight_rect(rect: Rect2) -> void:
	# Spotlight Control의 위치와 크기를 설정한다.
	# 실제 구현에서는 Shader 또는 여러 ColorRect를 조합하여
	# 사각형 구멍(hole)을 만든다.
	spotlight.position = rect.position
	spotlight.size = rect.size


## 툴팁 등장 애니메이션
func _animate_tooltip_in() -> void:
	tooltip_panel.modulate.a = 0.0
	tooltip_panel.scale = Vector2(0.9, 0.9)
	tooltip_panel.pivot_offset = tooltip_panel.size / 2.0
	var tw := create_tween().set_parallel(true)
	tw.tween_property(tooltip_panel, "modulate:a", 1.0, TOOLTIP_APPEAR_DURATION)
	tw.tween_property(tooltip_panel, "scale", Vector2.ONE, TOOLTIP_APPEAR_DURATION)\
		.set_ease(Tween.EASE_OUT).set_trans(Tween.TRANS_BACK)


## 다음 스텝으로 진행한다.
func _advance_step() -> void:
	_current_step += 1
	step_dismissed.emit()

	if _current_step >= _steps.size():
		_complete_sequence()
	else:
		_show_step(_current_step)


## 시퀀스 완료 처리
func _complete_sequence() -> void:
	_stop_hand_animation()
	visible = false
	guide_completed.emit()


## 확인 버튼 클릭
func _on_dismiss_pressed() -> void:
	_advance_step()
```

### 4.3 스포트라이트 효과 구현 (Shader)

스포트라이트(구멍 뚫린 딤)는 Shader로 구현한다.

```gdscript
## spotlight_shader.gdshader
## 중앙에 둥근 사각형 구멍이 있는 딤 효과
shader_type canvas_item;

uniform vec4 dim_color : source_color = vec4(0.0, 0.0, 0.0, 0.7);
uniform vec2 hole_position = vec2(0.5, 0.5);   // UV 좌표
uniform vec2 hole_size = vec2(0.3, 0.3);        // UV 크기
uniform float corner_radius = 0.02;

void fragment() {
    vec2 uv = UV;
    vec2 dist = abs(uv - hole_position) - hole_size * 0.5 + corner_radius;
    float d = length(max(dist, 0.0)) - corner_radius;
    float mask = smoothstep(0.0, 0.01, d);
    COLOR = vec4(dim_color.rgb, dim_color.a * mask);
}
```

### 5.2 Toast 알림 (간단한 알림)

화면 상단에서 내려와 2~3초 표시 후 사라지는 간단한 알림이다.

```gdscript
## toast_manager.gd (Autoload)
## 화면 상단 토스트 알림을 관리한다.
extends CanvasLayer

## --- 상수 ---
const TOAST_DURATION: float = 3.0
const SLIDE_DURATION: float = 0.3

## --- 노드 ---
@onready var toast_container: PanelContainer = $ToastContainer
@onready var toast_icon: TextureRect = $ToastContainer/HBoxContainer/Icon
@onready var toast_label: Label = $ToastContainer/HBoxContainer/Label

## --- 토스트 큐 ---
var _toast_queue: Array[Dictionary] = []
var _is_showing: bool = false


## 토스트를 표시한다.
func show_toast(message: String, icon: Texture2D = null) -> void:
	_toast_queue.append({"message": message, "icon": icon})
	if not _is_showing:
		_show_next_toast()


## 다음 토스트를 표시한다.
func _show_next_toast() -> void:
	if _toast_queue.is_empty():
		_is_showing = false
		return

	_is_showing = true
	var toast_data: Dictionary = _toast_queue.pop_front()

	toast_label.text = toast_data.message
	if toast_data.icon:
		toast_icon.texture = toast_data.icon
		toast_icon.visible = true
	else:
		toast_icon.visible = false

	# 슬라이드 인 (위에서 아래)
	toast_container.position.y = -toast_container.size.y
	toast_container.visible = true
	var tw := create_tween()
	tw.tween_property(toast_container, "position:y", 20.0, SLIDE_DURATION)\
		.set_ease(Tween.EASE_OUT).set_trans(Tween.TRANS_BACK)

	# 표시 유지
	tw.tween_interval(TOAST_DURATION)

	# 슬라이드 아웃
	tw.tween_property(toast_container, "position:y",
		-toast_container.size.y, SLIDE_DURATION)\
		.set_ease(Tween.EASE_IN).set_trans(Tween.TRANS_CUBIC)

	tw.tween_callback(func() -> void:
		toast_container.visible = false
		_show_next_toast()
	)
```

### 6.2 장기 부재 유저 (3일 이상)

마지막 플레이로부터 3일 이상 경과한 경우 환영 팝업을 표시한다.

```gdscript
## returning_user_checker.gd
## 장기 부재 유저를 감지하고 환영 메시지를 표시한다.
class_name ReturningUserChecker
extends RefCounted

## 부재로 간주하는 최소 일수
const ABSENCE_THRESHOLD_DAYS: int = 3


## 장기 부재 유저인지 확인하고, 해당하면 환영 팝업을 표시한다.
static func check_and_welcome() -> void:
	var last_play: String = SaveManager.get_data("last_play_date", "")
	if last_play.is_empty():
		return

	var today: String = Time.get_date_string_from_system()
	var days_absent: int = _calculate_days_between(last_play, today)

	if days_absent >= ABSENCE_THRESHOLD_DAYS:
		_show_welcome_back_popup(days_absent)

	# 오늘 날짜로 업데이트
	SaveManager.set_data("last_play_date", today)
	SaveManager.save()


## 두 날짜 사이의 일수를 계산한다.
static func _calculate_days_between(date1: String, date2: String) -> int:
	var dict1 := Time.get_datetime_dict_from_datetime_string(date1 + "T00:00:00", false)
	var dict2 := Time.get_datetime_dict_from_datetime_string(date2 + "T00:00:00", false)
	var unix1 := Time.get_unix_time_from_datetime_dict(dict1)
	var unix2 := Time.get_unix_time_from_datetime_dict(dict2)
	return int(absf(unix2 - unix1) / 86400.0)


## 환영 팝업을 표시한다.
static func _show_welcome_back_popup(days: int) -> void:
	var stage: int = SaveManager.get_current_stage()
	var streak: int = SaveManager.get_daily_streak()

	ScreenManager.open_popup("welcome_back", {
		"days_absent": days,
		"current_stage": stage,
		"message": "다시 오신 것을 환영합니다!",
		"sub_message": "Stage %d에서 계속하세요." % stage,
	})
```

### 6.3 Daily 스트릭 초기화

장기 부재 시 Daily Challenge 연속 스트릭이 초기화될 수 있다. 이 확인은 앱 시작 시 수행한다.

```gdscript
## daily_streak_validator.gd
## 앱 시작 시 Daily 스트릭의 유효성을 확인하고 필요하면 초기화한다.
class_name DailyStreakValidator
extends RefCounted


## 스트릭 유효성을 확인한다.
## 마지막 Daily 완료일로부터 2일 이상 경과하면 스트릭을 초기화한다.
static func validate() -> void:
	var last_daily: String = SaveManager.get_data("last_daily_date", "")
	if last_daily.is_empty():
		return

	var today: String = Time.get_date_string_from_system()
	var days_diff: int = ReturningUserChecker._calculate_days_between(last_daily, today)

	# 2일 이상 경과 시 스트릭 초기화
	# (어제 완료 = 1일 차이 = 유효, 그저께 = 2일 차이 = 초기화)
	if days_diff >= 2:
		var old_streak: int = SaveManager.get_daily_streak()
		if old_streak > 0:
			SaveManager.set_data("daily_streak", 0)
			SaveManager.save()
			if OS.is_debug_build():
				print("DailyStreakValidator: 스트릭 초기화 (%d → 0, %d일 경과)" % [
					old_streak, days_diff
				])
```

### 7.2 TutorialManager 구현

```gdscript
## tutorial_manager.gd (Autoload)
## 튜토리얼 진행 상태를 관리하고 가이드 오버레이를 제어한다.

## --- 시그널 ---

## --- 상수 ---
const TUTORIAL_STAGES: int = 3  # 튜토리얼 총 스테이지 수

## --- 상태 ---
var _is_in_tutorial: bool = false
var _current_tutorial_stage: int = 0
var _guide_overlay: Node = null  # GuideOverlay 인스턴스

## --- 가이드 오버레이 씬 ---
var _guide_overlay_scene: PackedScene = preload(
	"res://scenes/components/tutorial_overlay.tscn"
)


## 튜토리얼을 시작한다.
func start_tutorial() -> void:
	_is_in_tutorial = true
	_current_tutorial_stage = 1

	tutorial_started.emit()

	# 첫 번째 튜토리얼 스테이지 시작
	_start_tutorial_stage(1)


## 튜토리얼 스테이지를 시작한다.
func _start_tutorial_stage(stage: int) -> void:
	_current_tutorial_stage = stage
	var language_index: int = SaveManager.get_settings().get("language_index", 0)

	var grid_data: GridData = null
	var tutorial_config: Dictionary = {}

	match stage:
		1:
			grid_data = TutorialPuzzles.get_stage_1_puzzle(language_index)
			tutorial_config = {
				"show_guide": true,
				"guide_steps": _get_stage_1_guide_steps(language_index),
			}
		2:
			grid_data = TutorialPuzzles.get_stage_2_puzzle(language_index)
			tutorial_config = {
				"show_guide": true,
				"guide_steps": _get_stage_2_guide_steps(language_index),
			}
		3:
			# Stage 3은 랜덤 생성
			tutorial_config = TutorialPuzzles.get_stage_3_config()

	# GameController를 통해 스테이지 시작
	GameController.start_stage(stage, GameControllerClass.GameMode.CLASSIC, "animals")

	# GameScreen Push
	ScreenManager.push_screen("game", {
		"stage": stage,
		"grid_data": grid_data,
		"tutorial_config": tutorial_config,
		"is_tutorial": true,
	})


## 튜토리얼 스테이지 클리어 시 호출한다.

	if stage >= TUTORIAL_STAGES:
		# 모든 튜토리얼 완료
		_complete_tutorial()
	else:
		# 다음 튜토리얼 스테이지
		_start_tutorial_stage(stage + 1)


## 튜토리얼 완료 처리
func _complete_tutorial() -> void:
	_is_in_tutorial = false
	SaveManager.set_tutorial_completed(true)
	SaveManager.set_current_stage(4)  # 일반 게임 Stage 4부터 시작
	SaveManager.save()

	tutorial_completed.emit()

	if OS.is_debug_build():
		print("TutorialManager: 튜토리얼 완료!")


## 현재 튜토리얼 진행 중인지 확인한다.

	return _is_in_tutorial


## 현재 튜토리얼 스테이지 번호를 반환한다.
func get_current_tutorial_stage() -> int:

## 튜토리얼 모드에서의 특수 처리

func enter(data: Dictionary = {}) -> void:
	var is_tutorial: bool = data.get("is_tutorial", false)
	var tutorial_config: Dictionary = data.get("tutorial_config", {})

	# 격자 데이터 설정 (튜토리얼은 고정 데이터 사용 가능)

	# 튜토리얼 가이드 표시
	if is_tutorial and tutorial_config.has("guide_steps"):
		var steps: Array[Dictionary] = tutorial_config.guide_steps
		# 격자 영역 좌표를 가이드 스텝에 반영
		_inject_grid_coordinates(steps)
		TutorialManager.show_guide(steps)

	# 튜토리얼 모드 UI 조정
	if is_tutorial:
		hint_bar.visible = false  # 힌트 버튼 숨김
		# 일시정지 버튼 비활성화
		top_bar.get_node("SettingsButton").disabled = true


## 스테이지 클리어 처리
func _on_all_words_found() -> void:
	if TutorialManager.is_in_tutorial():
		# 튜토리얼 클리어: 간소화된 결과 표시 후 다음 튜토리얼 진행
		var stage: int = TutorialManager.get_current_tutorial_stage()
		_show_tutorial_clear_message(stage)
		await get_tree().create_timer(1.5).timeout
		TutorialManager.on_tutorial_stage_cleared(stage)
	else:
		# 일반 게임 클리어
		GameController.complete_stage()


## 튜토리얼 클리어 메시지 표시
func _show_tutorial_clear_message(stage: int) -> void:
	var lang: int = SaveManager.get_settings().get("language_index", 0)
	var messages: Array = []
	if lang == 0:
		messages = ["Great job!", "Well done!", "Excellent! You're ready!"]
	else:
		messages = ["잘했어요!", "훌륭해요!", "완벽해요! 준비가 되었어요!"]

	var idx: int = mini(stage - 1, messages.size() - 1)

	# 화면 중앙에 축하 메시지 표시
	var label := Label.new()
	label.text = messages[idx]
	label.horizontal_alignment = HORIZONTAL_ALIGNMENT_CENTER
	label.add_theme_font_size_override("font_size", 48)
	label.anchors_preset = Control.PRESET_CENTER
	add_child(label)

	# 팝업 애니메이션
	label.scale = Vector2.ZERO
	label.pivot_offset = label.size / 2.0
	var tw := create_tween()
	tw.tween_property(label, "scale", Vector2.ONE, 0.4)\
		.set_ease(Tween.EASE_OUT).set_trans(Tween.TRANS_ELASTIC)
	tw.tween_interval(1.0)
	tw.tween_property(label, "modulate:a", 0.0, 0.3)
	tw.tween_callback(func() -> void:
		label.queue_free()
	)
```
