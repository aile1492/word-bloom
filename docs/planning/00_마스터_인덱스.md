# Word Search Puzzle - 기획서 최종본 마스터 인덱스

| 항목 | 내용 |
|------|------|
| **버전** | 1.0 |
| **작성일** | 2026-03-11 |
| **엔진** | Godot 4.6 Stable + GDScript |
| **타겟 플랫폼** | Android, iOS, Android TV, Fire TV |

---

## 1. 프로젝트 개요

클래식 단어 찾기(Word Search) 퍼즐 게임으로, 스테이지 기반 진행 시스템과 8방향 단어 배치를 핵심 메커닉으로 한다. 영어(EN)와 한국어(KO) 이중 언어를 지원하며, Classic·TimeAttack·Daily Challenge·Bonus·Marathon 등 5개 게임 모드를 제공한다. 듀얼 재화(코인/젬) 경제, 광고, IAP, 구독 기반의 수익화 시스템과 일일 보상·스트릭·업적·컬렉션 등의 리텐션 시스템을 갖춘다. 타겟 유저는 전 연령대의 캐주얼 퍼즐 플레이어이며, Godot 4.6 + GDScript로 개발하여 모바일(Android/iOS) 및 TV(Android TV/Fire TV) 4개 플랫폼에 출시한다.

---

## 2. 문서 작성 원칙

- **언어**: 모든 문서는 한국어로 작성하되, 영문 기술 용어(Node, Signal, Resource 등)는 원문을 유지한다.
- **엔진 기준**: 모든 기술 내용은 Godot 4.6 + GDScript 네이티브 기준이며, Unity 참조를 포함하지 않는다.
- **독립 가독성**: 각 문서는 단독으로 읽을 수 있도록 작성하되, 관련 문서 간 상호 참조를 명시한다.
- **코드 포함**: 핵심 알고리즘에는 GDScript 의사코드(pseudocode)를 포함한다.
- **포팅 추적**: 기존 C# 레퍼런스 파일이 있는 경우, 포팅 추적을 위해 원본 파일명을 문서 내에 기록한다.

## 7. 읽기 가이드

```
APP_02 용어집
  → P00_01 프로젝트 셋업 가이드
    → P00_02 코딩 컨벤션 및 AI 규칙
      → APP_01 기술 스택 총괄
        → P01_01 → P01_02 → P01_03 (핵심 게임플레이)
          → CC_01 다국어 지원
            → P02_01 → P02_02 → P02_03 → P02_04 (게임 플로우)
              → P03_01 (난이도) → P04_01 (힌트) → P05_01 (모드)
                → P06_01 (수익화) → P07_01 (리텐션)
                  → P08_01~03 (UI/UX) → CC_02 (TV)
                    → P09_01 (소셜) → P10_01 (빌드)
                      → CC_04 (테스트) → CC_05 (분석)
```

```

  → P08_01 UI 디자인 시스템
```
