## 2. 사운드 디자인 철학

### 2.1 핵심 원칙

단어 찾기 퍼즐은 **집중력이 핵심**인 게임이다. 사운드 디자인은 플레이어의 몰입을 보조하되, 방해하지 않는 방향으로 설계한다.

| 원칙 | 설명 | 적용 예시 |
|------|------|-----------|
| **집중 방해 최소화** | 반복되는 SFX는 짧고 부드럽게 유지한다. 날카로운 고음, 큰 볼륨의 갑작스러운 사운드를 지양한다. | `cell_select`는 0.1초 이내의 소프트 틱 사운드 |
| **긍정적 피드백 강화** | 올바른 동작에 대해 명확하고 기분 좋은 청각 피드백을 제공한다. 부정적 피드백은 최소화한다. | `word_found`는 밝은 차임벨, `word_wrong`은 부드러운 저음 |
| **일관된 톤** | 모든 사운드가 하나의 청각적 세계관을 공유한다. 장르 혼합을 피하고, 캐주얼/미니멀 톤을 유지한다. | BGM과 SFX 모두 동일한 음색 팔레트에서 선택 |
| **점진적 보상감** | 콤보 시스템을 통해 연속 성공 시 사운드가 상승하며, 달성감을 청각적으로 전달한다. | 콤보별 피치 상승으로 "음계가 올라가는" 느낌 |

### 2.2 톤 가이드라인

- **장르**: 캐주얼 퍼즐 (밝고 가벼운 앰비언트)
- **악기 톤**: 마림바, 실로폰, 글로켄슈필, 소프트 신스
- **분위기**: 따뜻하고 편안한, 카페에서 퍼즐을 푸는 느낌
- **피해야 할 것**: 8bit 사운드, 무거운 베이스, 공포/긴장 효과음, 보컬

## 4. 콤보 피치 시스템

### 4.1 개요

플레이어가 단어를 연속으로 발견하면(10초 이내), 사운드의 피치가 점진적으로 올라가며 달성감을 청각적으로 강화한다. 이는 음악에서 "음계가 상승하는" 느낌을 준다.

### 4.2 콤보 규칙

| 항목 | 값 |
|------|-----|
| **콤보 인정 시간** | 이전 단어 발견 후 10초 이내 |
| **콤보 리셋 조건** | 10초 타이머 만료 |
| **기본 피치 (base_pitch)** | 1.0 |
| **`word_found` 피치 증가량** | 콤보당 +0.1 |
| **`cell_select` 피치 증가량** | 콤보당 +0.05 |
| **최대 피치 오프셋** | +0.3 (base_pitch 1.0 + 0.3 = 1.3) |
| **최대 콤보 단계** | 3 (0.3 / 0.1) |
| **리셋 전환** | 즉시 복귀가 아닌 0.3초에 걸쳐 base_pitch로 Tween |

### 4.3 피치 계산 공식

```
combo_pitch_offset = min(combo_count * pitch_per_combo, MAX_PITCH_OFFSET)
final_pitch = BASE_PITCH + combo_pitch_offset
```

### 4.4 콤보별 피치 테이블

| 콤보 | `word_found` 피치 | `cell_select` 피치 | 체감 |
|:----:|:-----------------:|:------------------:|------|
| 0 | 1.00 | 1.00 | 기본 |
| 1 | 1.10 | 1.05 | 약간 밝아짐 |
| 2 | 1.20 | 1.10 | 명확히 올라감 |
| 3+ | 1.30 | 1.15 | 최대치, 경쾌하게 |

### 4.5 GDScript 구현

```gdscript
## combo_pitch_tracker.gd (audio_manager.gd 내부 또는 별도 클래스)
## 콤보 기반 피치 추적기. 단어 연속 발견 시 피치를 점진적으로 올린다.
class_name ComboPitchTracker
extends RefCounted

## 피치 관련 상수
const BASE_PITCH: float = 1.0
const MAX_PITCH_OFFSET: float = 0.3
const WORD_FOUND_PITCH_STEP: float = 0.1
const CELL_SELECT_PITCH_STEP: float = 0.05
const COMBO_TIMEOUT: float = 10.0

## 현재 콤보 카운트
var combo_count: int = 0

## 마지막 단어 발견 시각 (Time.get_ticks_msec 기준)
var _last_word_time_ms: int = 0


## 단어 발견 시 호출. 콤보를 갱신하고 word_found 피치를 반환한다.
func on_word_found() -> float:
    var now_ms: int = Time.get_ticks_msec()
    var elapsed_sec: float = (now_ms - _last_word_time_ms) / 1000.0

    if _last_word_time_ms > 0 and elapsed_sec <= COMBO_TIMEOUT:
        combo_count = mini(combo_count + 1, int(MAX_PITCH_OFFSET / WORD_FOUND_PITCH_STEP))
    else:
        combo_count = 0

    _last_word_time_ms = now_ms
    return BASE_PITCH + combo_count * WORD_FOUND_PITCH_STEP


## 셀 선택 시 호출. 현재 콤보에 따른 cell_select 피치를 반환한다.
func get_cell_select_pitch() -> float:
    # 콤보 타임아웃 체크
    var now_ms: int = Time.get_ticks_msec()
    var elapsed_sec: float = (now_ms - _last_word_time_ms) / 1000.0

    if _last_word_time_ms > 0 and elapsed_sec > COMBO_TIMEOUT:
        combo_count = 0

    var offset: float = minf(combo_count * CELL_SELECT_PITCH_STEP, MAX_PITCH_OFFSET)
    return BASE_PITCH + offset


## 콤보를 강제 리셋한다 (스테이지 종료 등).
func reset() -> void:
    combo_count = 0
    _last_word_time_ms = 0
```

| 화면 / 상태 | BGM ID | 전환 방식 |
|-------------|--------|-----------|
| 타이틀 화면 | `bgm_main` | Fade In (1초) |
| 홈 화면 | `bgm_main` | 유지 (같은 BGM) |
| 스테이지 선택 | `bgm_main` | 유지 |
| Classic / Marathon 플레이 | `bgm_play` | Crossfade (1초) |
| Time Attack / Daily 플레이 | `bgm_daily` | Crossfade (1초) |
| 결과 화면 | `bgm_result` | Crossfade (1초) |
| 설정 팝업 | (현재 BGM 유지) | 볼륨 -6dB 덕킹 |
| 앱 백그라운드 | (현재 BGM 유지) | Fade Out -> Mute |
| 앱 포그라운드 복귀 | (현재 BGM 유지) | Fade In (0.5초) |

### 5.3 BGM 크로스페이드 메커니즘

두 개의 `AudioStreamPlayer` Node를 번갈아 사용하여 끊김 없는 전환을 구현한다.

```
[BGM Player A] ──── Volume: 0dB ────────→ Fade Out (1초) → 정지
[BGM Player B] ──── Volume: -80dB ──→ Fade In (1초) → 0dB (새 BGM 재생)
```

다음 전환 시에는 A와 B의 역할이 반대가 된다.

### 5.4 루프 포인트 설정 방법

Godot에서 .ogg 파일의 심리스 루프를 설정하는 절차이다.

1. **파일 Import**: .ogg 파일을 `res://assets/audio/bgm/`에 배치한다.
2. **Import 탭 설정**:
   - FileSystem 패널에서 .ogg 파일 선택
   - Inspector > Import 탭 열기
   - `loop` = `true` 체크
   - `loop_offset` = `0.0` (시작점부터 루프) 또는 인트로 이후 시점(초) 입력
   - "Reimport" 클릭
3. **검증**: AudioStreamPlayer에 할당 후 재생하여 루프 이음새에 클릭 노이즈가 없는지 확인한다.

> **주의**: .ogg (Ogg Vorbis) 포맷만 Godot에서 네이티브 루프를 지원한다. .mp3는 디코딩 특성상 루프 시작/끝에 무음 갭이 발생하므로 BGM에 사용하지 않는다.

### 5.5 BGM 볼륨 덕킹

팝업이 열리거나 중요한 SFX(팡파레 등)가 재생될 때 BGM 볼륨을 일시적으로 낮춰 청각적 명확성을 확보한다.

| 상황 | 덕킹 레벨 | 복귀 시간 |
|------|-----------|-----------|
| 설정 팝업 열림 | -6dB | 팝업 닫힘 후 0.3초 |
| `stage_clear` 팡파레 | -12dB | 팡파레 종료 후 0.5초 |
| `achievement_unlock` | -6dB | SFX 종료 후 0.3초 |
| `new_record` | -12dB | SFX 종료 후 0.5초 |

## 6. AudioBus 설계

### 6.1 Bus 레이아웃

```
Master (idx: 0)
│   └── Volume: 0dB
│
├── Music (idx: 1)
│   └── Volume: -6dB (default)
│   └── 용도: 모든 BGM
│   └── Effect: (없음, 필요 시 LowPassFilter 추가 가능)
│
├── SFX (idx: 2)
│   └── Volume: 0dB (default)
│   └── 용도: 게임플레이 효과음 (cell_select, word_found, stage_clear 등)
│
└── UI (idx: 3)
    └── Volume: -3dB (default)
    └── 용도: UI 인터랙션 사운드 (button_click, popup_open)
```

### 6.2 Bus 설정 방법

Godot 에디터에서 하단 `Audio` 탭을 열어 위 구조대로 Bus를 생성한다. 또는 `default_bus_layout.tres` 파일을 직접 편집할 수 있다.

### 6.3 런타임 볼륨 제어

Settings 화면의 슬라이더와 AudioBus를 연동한다.

```gdscript
## 볼륨 값(0.0 ~ 1.0)을 dB로 변환하여 Bus에 적용한다.
## value: 0.0 = 음소거(-80dB), 1.0 = 최대(0dB)
func _set_bus_volume(bus_name: String, value: float) -> void:
    var bus_idx: int = AudioServer.get_bus_index(bus_name)
    if bus_idx == -1:
        push_warning("AudioBus not found: %s" % bus_name)
        return

    if value <= 0.0:
        AudioServer.set_bus_mute(bus_idx, true)
    else:
        AudioServer.set_bus_mute(bus_idx, false)
        var db: float = linear_to_db(value)
        AudioServer.set_bus_volume_db(bus_idx, db)
```

## 7. audio_manager.gd 구현

### 7.1 설계 개요

`AudioManager`는 **Autoload Singleton**으로 등록하여 모든 씬에서 전역 접근이 가능하게 한다. 주요 책임은 다음과 같다.

- 볼륨 설정 연동
- 앱 포커스 변화에 따른 자동 뮤트

`Project Settings > Globals > Autoload`에서 등록한다.

```gdscript
        add_child(player)

# ──────────────────────────────────────────
# SFX 재생
# ──────────────────────────────────────────

## SFX를 재생한다.
## sfx_name: _sfx_library에 등록된 이벤트 ID (예: "word_found")
## pitch_scale: 피치 스케일 (기본 1.0, 콤보 시스템에서 조절)
func play_sfx(sfx_name: String, pitch_scale: float = 1.0) -> void:
    if not _sfx_library.has(sfx_name):
        push_warning("AudioManager: Unknown SFX '%s'" % sfx_name)
        return

    var entry: Dictionary = _sfx_library[sfx_name]
    var stream: AudioStream = _load_sfx_stream(entry["file"])
    if stream == null:
        return

    var player: AudioStreamPlayer = _get_available_sfx_player()
    if player == null:
        # 풀이 모두 사용 중이면 가장 오래된 것을 재활용
        player = _sfx_pool[0]
        player.stop()

    player.stream = stream
    player.volume_db = linear_to_db(entry["volume"])
    player.pitch_scale = pitch_scale
    player.bus = entry["bus"]
    player.play()


## 콤보를 반영한 cell_select SFX를 재생한다.
func play_cell_select() -> void:
    var pitch: float = _combo_tracker.get_cell_select_pitch()
    play_sfx("cell_select", pitch)


## 콤보를 반영한 word_found SFX를 재생한다.
## 콤보 카운트를 갱신하고, 해당 피치로 재생한다.
func play_word_found() -> void:
    var pitch: float = _combo_tracker.on_word_found()
    play_sfx("word_found", pitch)


## 등급에 따른 grade_reveal SFX를 재생한다.
## grade: "s", "a", "b", "c" 중 하나
func play_grade_reveal(grade: String) -> void:

    play_sfx(sfx_name)


## 콤보 피치 추적기를 리셋한다. 스테이지 종료 시 호출한다.
func reset_combo() -> void:
    _combo_tracker.reset()


## SFX 스트림을 로드한다. 캐시를 먼저 확인하고, 없으면 로드 후 캐시에 저장한다.
func _load_sfx_stream(file_name: String) -> AudioStream:
    var path: String = SFX_PATH_PREFIX + file_name

    if _stream_cache.has(path):
        return _stream_cache[path]

    if not ResourceLoader.exists(path):
        push_warning("AudioManager: SFX file not found at '%s'" % path)
        return null

    var stream: AudioStream = load(path)
    _stream_cache[path] = stream
    return stream


## 풀에서 사용 가능한 (재생 중이 아닌) SFX Player를 반환한다.
## 모두 사용 중이면 null을 반환한다.
func _get_available_sfx_player() -> AudioStreamPlayer:
    for player in _sfx_pool:
        if not player.playing:
            return player
    return null


# ──────────────────────────────────────────
# BGM 재생
# ──────────────────────────────────────────

## BGM을 재생한다. 이미 같은 BGM이 재생 중이면 무시한다.
## bgm_name: BGM ID (예: "bgm_main", "bgm_play")
## crossfade: 크로스페이드 시간 (초). 0이면 즉시 전환.
func play_bgm(bgm_name: String, crossfade: float = DEFAULT_CROSSFADE_TIME) -> void:
    # 같은 BGM이면 무시
    if bgm_name == _current_bgm_id and _active_bgm_player.playing:
        return

    var path: String = BGM_PATH_PREFIX + bgm_name + ".ogg"
    if not ResourceLoader.exists(path):
        push_warning("AudioManager: BGM file not found at '%s'" % path)
        return

    var stream: AudioStream = load(path)
    _current_bgm_id = bgm_name

    # 비활성 Player를 새 BGM에 사용
    var new_player: AudioStreamPlayer = _get_inactive_bgm_player()
    var old_player: AudioStreamPlayer = _active_bgm_player

    new_player.stream = stream
    new_player.volume_db = MUTE_DB
    new_player.play()

    if crossfade > 0.0:
        _crossfade(old_player, new_player, crossfade)
    else:
        old_player.stop()
        old_player.volume_db = MUTE_DB
        new_player.volume_db = _pre_duck_volume_db

    _active_bgm_player = new_player


## BGM을 정지한다.
## fade_out: 페이드아웃 시간 (초). 0이면 즉시 정지.
func stop_bgm(fade_out: float = DEFAULT_CROSSFADE_TIME) -> void:
    if not _active_bgm_player.playing:
        return

    _current_bgm_id = ""

    if fade_out > 0.0:
        var tween: Tween = create_tween()
        tween.tween_property(
            _active_bgm_player, "volume_db",
            MUTE_DB, fade_out
        ).set_ease(Tween.EASE_IN)
        tween.tween_callback(_active_bgm_player.stop)
    else:
        _active_bgm_player.stop()
        _active_bgm_player.volume_db = MUTE_DB


## BGM 크로스페이드를 실행한다.
func _crossfade(
    old_player: AudioStreamPlayer,
    new_player: AudioStreamPlayer,
    duration: float
) -> void:
    var tween: Tween = create_tween()
    tween.set_parallel(true)

    # 기존 BGM 페이드아웃
    tween.tween_property(
        old_player, "volume_db",
        MUTE_DB, duration
    ).set_ease(Tween.EASE_IN)

    # 새 BGM 페이드인
    tween.tween_property(
        new_player, "volume_db",
        _pre_duck_volume_db, duration
    ).set_ease(Tween.EASE_OUT)

    # 페이드아웃 완료 후 이전 Player 정지
    tween.set_parallel(false)
    tween.tween_callback(old_player.stop)


## 현재 비활성 BGM Player를 반환한다.
func _get_inactive_bgm_player() -> AudioStreamPlayer:
    if _active_bgm_player == _bgm_player_a:
        return _bgm_player_b
    return _bgm_player_a


# ──────────────────────────────────────────
# BGM 덕킹
# ──────────────────────────────────────────

## BGM 볼륨을 일시적으로 낮춘다 (덕킹).
## duck_db: 낮출 볼륨량 (음수 값, 예: -6.0)
## duration: 전환 시간 (초)
func duck_bgm(duck_db: float, duration: float = 0.3) -> void:
    if _ducking_tween and _ducking_tween.is_valid():
        _ducking_tween.kill()

    var target_db: float = _pre_duck_volume_db + duck_db
    _ducking_tween = create_tween()
    _ducking_tween.tween_property(
        _active_bgm_player, "volume_db",
        target_db, duration
    )


## BGM 덕킹을 해제하고 원래 볼륨으로 복귀한다.
## duration: 복귀 시간 (초)
func unduck_bgm(duration: float = 0.3) -> void:
    if _ducking_tween and _ducking_tween.is_valid():
        _ducking_tween.kill()

    _ducking_tween = create_tween()
    _ducking_tween.tween_property(
        _active_bgm_player, "volume_db",
        _pre_duck_volume_db, duration
    )


# ──────────────────────────────────────────
# 볼륨 설정
# ──────────────────────────────────────────

## Music 볼륨을 설정한다.
## value: 0.0 (음소거) ~ 1.0 (최대)
func set_music_volume(value: float) -> void:
    _set_bus_volume(BUS_MUSIC, value)
    # 덕킹 기준 볼륨도 갱신
    if value <= 0.0:
        _pre_duck_volume_db = MUTE_DB
    else:
        _pre_duck_volume_db = linear_to_db(value)


## SFX 볼륨을 설정한다.
## value: 0.0 (음소거) ~ 1.0 (최대)
func set_sfx_volume(value: float) -> void:
    _set_bus_volume(BUS_SFX, value)
    _set_bus_volume(BUS_UI, value)


## Master 볼륨을 설정한다.
## value: 0.0 (음소거) ~ 1.0 (최대)
func set_master_volume(value: float) -> void:
    _set_bus_volume(BUS_MASTER, value)


## AudioBus 볼륨을 설정하는 내부 헬퍼이다.
func _set_bus_volume(bus_name: StringName, value: float) -> void:
    var bus_idx: int = AudioServer.get_bus_index(bus_name)
    if bus_idx == -1:
        push_warning("AudioManager: AudioBus not found: %s" % bus_name)
        return

    if value <= 0.0:
        AudioServer.set_bus_mute(bus_idx, true)
    else:
        AudioServer.set_bus_mute(bus_idx, false)
        AudioServer.set_bus_volume_db(bus_idx, linear_to_db(value))


# ──────────────────────────────────────────
# 앱 포커스 처리
# ──────────────────────────────────────────

## 앱이 포커스를 얻었을 때 호출한다.
func _on_app_focus_entered() -> void:
    _app_focused = true
    # BGM이 재생 중이었으면 페이드인으로 복귀
    if _active_bgm_player.stream and not _active_bgm_player.playing:
        _active_bgm_player.play()
    if _active_bgm_player.playing:
        var tween: Tween = create_tween()
        tween.tween_property(
            _active_bgm_player, "volume_db",
            _pre_duck_volume_db, 0.5
        )


## 앱이 포커스를 잃었을 때 호출한다.
func _on_app_focus_exited() -> void:
    _app_focused = false
    # 모든 사운드를 페이드아웃
    if _active_bgm_player.playing:
        var tween: Tween = create_tween()
        tween.tween_property(
            _active_bgm_player, "volume_db",
            MUTE_DB, 0.3
        )
    # SFX 즉시 정지
    for player in _sfx_pool:
        if player.playing:
            player.stop()


# ──────────────────────────────────────────
# 유틸리티
# ──────────────────────────────────────────

## 현재 BGM ID를 반환한다. 재생 중이 아니면 빈 문자열이다.
func get_current_bgm() -> String:
    return _current_bgm_id


## BGM이 재생 중인지 확인한다.
func is_bgm_playing() -> bool:
    return _active_bgm_player.playing


## 모든 사운드를 즉시 정지한다.
func stop_all() -> void:
    stop_bgm(0.0)
    for player in _sfx_pool:
        if player.playing:
            player.stop()
    _current_bgm_id = ""
    _combo_tracker.reset()
```

```gdscript
    AudioManager.reset_combo()
    # 팡파레 종료 후 덕킹 해제 (약 1.5초 후)
    await get_tree().create_timer(1.5).timeout
    AudioManager.unduck_bgm(0.5)


## ── 화면 전환 (P02_01 ScreenManager) ──
## 게임플레이 화면 진입
func _on_screen_changed(screen_name: String) -> void:
    match screen_name:
        "home", "title", "stage_select":
            AudioManager.play_bgm("bgm_main")
        "game_classic", "game_marathon":
            AudioManager.play_bgm("bgm_play")
        "game_timeattack", "game_daily":
            AudioManager.play_bgm("bgm_daily")
        "result":
            AudioManager.play_bgm("bgm_result")

## ── 설정 화면 ──
```

## 9. 파일 포맷 및 최적화

### 9.1 포맷 규격

| 항목 | SFX | BGM |
|------|-----|-----|
| **포맷** | .ogg (Ogg Vorbis) | .ogg (Ogg Vorbis) |
| **채널** | 모노 (Mono) | 스테레오 (Stereo) |
| **샘플레이트** | 44,100 Hz | 44,100 Hz |
| **비트 깊이** | 16-bit | 16-bit |
| **루프** | 불필요 | 필수 (Godot Import 설정) |
| **압축 품질** | Vorbis Quality 3~5 | Vorbis Quality 5~6 |

### 9.2 .mp3 사용 금지 이유

| 문제 | 설명 |
|------|------|
| **루프 갭** | .mp3 디코딩 시 프레임 패딩으로 인해 루프 시작/끝에 무음 갭이 발생한다. BGM의 심리스 루프가 불가능하다. |
| **라이선스** | .mp3 특허는 만료되었으나, .ogg가 오픈소스 표준으로 Godot에서 더 나은 지원을 받는다. |
| **Import 설정** | Godot의 .ogg Import에서 `loop`, `loop_offset` 파라미터를 직접 지원하지만, .mp3는 제한적이다. |

### 9.3 용량 목표

| 카테고리 | 파일 수 | 개당 목표 | 합계 목표 |
|----------|:-------:|:---------:|:---------:|
| SFX | 19개 | ~30KB | ~570KB |
| BGM | 4개 | ~800KB | ~3.2MB |
| **전체** | **23개** | - | **< 5MB** |

### 9.4 Godot Import 최적화 설정

각 파일의 Import 설정을 조정하여 용량을 최적화한다.

```
[SFX 파일 Import 설정]
- Resource Type: AudioStreamOggVorbis
- Loop: false (SFX는 루프 불필요)

[BGM 파일 Import 설정]
- Resource Type: AudioStreamOggVorbis
- Loop: true
- Loop Offset: 0.0 (또는 인트로 이후 시점)
```

### 9.5 오디오 전처리 파이프라인

에셋 다운로드 후 프로젝트에 통합하기 전 수행하는 전처리 단계이다.

```
1. [다운로드] CC0 에셋 다운로드
2. [포맷 변환] .wav/.mp3 → .ogg (Audacity 또는 ffmpeg 사용)
3. [채널 변환] SFX: 스테레오 → 모노 다운믹스
4. [트리밍] 불필요한 앞뒤 무음 구간 제거
5. [노멀라이즈] 피크 기준 -1dB로 정규화
6. [루프 편집] BGM: 루프 시작/끝 지점의 파형이 매끄럽게 이어지도록 편집
7. [Export] .ogg (Vorbis Quality 설정에 맞춰 내보내기)
8. [배치] res://assets/audio/sfx/ 또는 res://assets/audio/bgm/에 배치
9. [Import 확인] Godot에서 Import 설정 확인 및 Reimport
```

**ffmpeg 예시 명령어**:

```bash
# WAV → OGG 변환 (SFX, 모노, 품질 4)
ffmpeg -i input.wav -ac 1 -c:a libvorbis -q:a 4 sfx_tick.ogg

# WAV → OGG 변환 (BGM, 스테레오, 품질 6)
ffmpeg -i input.wav -ac 2 -c:a libvorbis -q:a 6 bgm_main.ogg
```

## 10. 접근성 고려

### 10.1 시각적 피드백 동반 원칙

모든 SFX 이벤트에는 반드시 대응하는 시각적 피드백이 존재해야 한다. 소리를 꺼도 게임 진행에 지장이 없어야 한다.

| SFX 이벤트 | 시각적 대응 피드백 |
|------------|---------------------|
| `cell_select` | 셀 하이라이트 색상 변경 (P08_01 참조) |
| `word_found` | 단어 라인 고정 + 색상 변경 + 파티클 이펙트 |
| `word_wrong` | 선택 라인 빨간색 플래시 후 해제 |
| `stage_clear` | 클리어 배너 애니메이션 + 별/파티클 연출 |
| `stage_fail` | 시간 초과 배너 + 화면 어둡게 전환 |
| `timer_warning` | 타이머 UI 빨간색 깜빡임 |
| `hint_activate` | 힌트 대상 셀 글로우 이펙트 |
| `coin_earn` | 코인 아이콘 바운스 + 숫자 카운트업 |
| `achievement_unlock` | 업적 토스트 팝업 슬라이드 |
| `new_record` | "New Record!" 텍스트 애니메이션 |

### 10.2 진동 피드백 (모바일)

모바일 플랫폼에서는 중요한 이벤트에 햅틱 피드백을 제공한다.

| 이벤트 | 진동 패턴 | 강도 | 길이 |
|--------|-----------|:----:|:----:|
| `word_found` | 단일 짧은 진동 | 약 | 50ms |
| `stage_clear` | 더블 진동 | 중 | 100ms + 50ms + 100ms |
| `word_wrong` | 단일 매우 짧은 진동 | 약 | 30ms |
| `new_record` | 트리플 진동 | 강 | 80ms + 40ms + 80ms + 40ms + 80ms |

GDScript에서 진동을 호출하는 방법이다.

```gdscript
## vibration_helper.gd
## 모바일 플랫폼의 진동 피드백 헬퍼이다.

## 짧은 진동 (단어 발견 등)
static func vibrate_light() -> void:
    if OS.has_feature("mobile"):
        Input.vibrate_handheld(50)


## 중간 진동 (스테이지 클리어 등)
static func vibrate_medium() -> void:
    if OS.has_feature("mobile"):
        Input.vibrate_handheld(100)


## 강한 진동 (신기록 등)
static func vibrate_strong() -> void:
    if OS.has_feature("mobile"):
        Input.vibrate_handheld(200)
```

> **참고**: Godot 4.x의 `Input.vibrate_handheld()`은 단일 진동만 지원한다. 패턴 진동(더블, 트리플)을 구현하려면 Timer를 이용하여 연속 호출해야 한다.

### 10.3 설정 화면 음량 컨트롤

| 설정 항목 | UI 위젯 | 값 범위 | 기본값 | 저장 키 |
|-----------|---------|---------|:------:|---------|
| 음악 볼륨 | HSlider + Toggle | 0.0 ~ 1.0 | 0.7 | `settings/music_volume` |
| 효과음 볼륨 | HSlider + Toggle | 0.0 ~ 1.0 | 0.8 | `settings/sfx_volume` |
| 진동 | CheckBox | on/off | on | `settings/vibration_enabled` |

- Toggle은 슬라이더 옆에 배치하여 빠르게 음소거/해제할 수 있게 한다.
- 볼륨 값은 `ConfigFile`(P02_03 저장/로드 시스템)을 통해 `user://settings.cfg`에 영속 저장한다.
- 앱 시작 시 저장된 볼륨을 `AudioManager`에 적용한다.

```gdscript
## settings_manager.gd (관련 부분 발췌)

func _apply_audio_settings() -> void:
    var music_vol: float = _config.get_value("settings", "music_volume", 0.7)
    var sfx_vol: float = _config.get_value("settings", "sfx_volume", 0.8)

    AudioManager.set_music_volume(music_vol)
    AudioManager.set_sfx_volume(sfx_vol)
```
