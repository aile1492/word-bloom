## 2. 디자인 원칙

### 2.1 핵심 원칙 5가지

| # | 원칙 | 설명 |
|:-:|------|------|
| 1 | **미니멀(Minimal)** | 불필요한 장식 요소를 배제하고, 격자와 단어 목록에 시각적 집중을 유도한다. |
| 2 | **집중 방해 최소(Low Distraction)** | 게임 플레이 중 UI 요소는 최소화한다. 헤더와 힌트 바만 표시하며 나머지는 격자 영역에 집중한다. |
| 3 | **터치 친화(Touch-first)** | 모든 인터랙티브 요소의 최소 터치 타겟은 44dp x 44dp (Android Material Guidelines)를 준수한다. |
| 4 | **TV 대응(TV-ready)** | D-pad Focus Navigation을 모든 화면에서 지원한다. 포커스 링(Focus Ring)은 명확한 시각적 피드백을 제공한다. |
| 5 | **접근성 고려(Accessible)** | WCAG 2.1 AA 기준 색상 대비(4.5:1 이상)를 준수하며, 폰트 크기 조절 옵션을 제공한다. |

### 2.2 디자인 키워드

```
깔끔 · 경쾌 · 둥근 모서리 · 밝은 톤 · 카드 UI · 마이크로 인터랙션
```

### 2.3 플랫폼별 디자인 차이

| 요소 | 모바일(Mobile) | TV (Lean-back) |
|------|---------------|----------------|
| 입력 방식 | 터치(드래그/탭) | D-pad + Select 버튼 |
| 기본 테마 | 라이트 테마 | 다크 테마 (거실 환경 최적화) |
| 폰트 스케일 | 1.0x (기준) | 1.7x ~ 1.8x |
| 셀 크기 | min 40dp | 64dp |
| 여백 | 기준값 | 기준값 x 1.5 |
| 안전 영역 | 없음 (OS 시스템 바 고려) | 화면 가장자리 5% (96px @1920x1080) |
| 하단 탭 바 | 표시 | 사이드 네비게이션으로 전환 |
| 호버 상태 | 없음 | 없음 (포커스로 대체) |

### 3.3 Godot Theme .tres에서의 색상 정의

Godot의 Theme Resource는 각 Control Type별로 Color, StyleBox, Font 등을 지정한다. 색상 팔레트를 Theme 외부의 GDScript Resource로 관리하고, Theme 생성 시 참조하는 방식을 권장한다.

```gdscript
# theme_colors.gd
# 색상 상수를 한 곳에서 관리한다. Theme .tres 생성 스크립트에서 참조한다.
class_name ThemeColors
extends RefCounted

# --- Light Theme ---
const LIGHT_BG := Color(0.961, 0.961, 0.961)          # #F5F5F5
const LIGHT_GRID_BG := Color.WHITE                      # #FFFFFF
const LIGHT_TEXT_PRIMARY := Color(0.2, 0.2, 0.2)       # #333333
const LIGHT_TEXT_SECONDARY := Color(0.459, 0.459, 0.459) # #757575
const LIGHT_SELECTING := Color(1.0, 0.835, 0.310)      # #FFD54F

const LIGHT_FOUND_COLORS: Array[Color] = [
    Color(0.298, 0.686, 0.314),  # #4CAF50
    Color(0.129, 0.588, 0.953),  # #2196F3
    Color(1.0, 0.341, 0.133),    # #FF5722
    Color(0.612, 0.153, 0.690),  # #9C27B0
    Color(0.0, 0.737, 0.831),    # #00BCD4
    Color(0.914, 0.118, 0.388),  # #E91E63
]

const LIGHT_HINT := Color(1.0, 0.561, 0.0)             # #FF8F00
const LIGHT_FALSE_LEAD := Color(0.459, 0.459, 0.459)   # #757575

# --- Dark Theme ---
const DARK_BG := Color(0.102, 0.102, 0.180)            # #1A1A2E

const DARK_SELECTING := Color(1.0, 0.835, 0.310)        # #FFD54F
```

### 4.3 TV 스케일링 규칙

TV 환경에서는 시청 거리(2~3m)를 고려하여 모바일 대비 **1.7x ~ 1.8x** 스케일링을 적용한다.

```gdscript
# layout_manager.gd 내 TV 폰트 스케일 적용
const TV_FONT_SCALE: float = 1.8
const TV_CELL_SCALE: float = 1.6

func get_font_size(base_size: int) -> int:
    if is_tv():
        return int(base_size * TV_FONT_SCALE)
    return base_size
```

### 4.4 Godot에서의 폰트 크기 설정

```
Theme Editor:
  Label > font_sizes > font_size: 16    (Body 기준)
  Label > fonts > font: NotoSansKR-Regular.ttf

  Button > font_sizes > font_size: 16
  Button > fonts > font: NotoSansKR-Bold.ttf
```

개별 Label에서 Theme Override로 크기를 재정의:
```
Label (H1):
  Theme Overrides > Font Sizes > font_size: 28
  Theme Overrides > Fonts > font: NotoSansKR-Bold.ttf
```

```gdscript
# letter_cell.gd
extends Button

enum CellState { NORMAL, SELECTED, FOUND, HINT, FALSE_LEAD }

@export var grid_x: int = 0
@export var grid_y: int = 0

@onready var background: Panel = $Background
@onready var letter_label: Label = $Background/LetterLabel
@onready var focus_indicator: Panel = $FocusIndicator

var found_color_index: int = -1  # Found 상태일 때 색상 인덱스 (0~5)
var letter: String = ""

func set_letter(value: String) -> void:
    letter = value
    letter_label.text = value

func set_state(new_state: CellState, color_idx: int = -1) -> void:
    current_state = new_state
    found_color_index = color_idx
    _update_visual()

func _update_visual() -> void:
    var theme_mgr = UIThemeManager  # Autoload 참조
    var style: StyleBoxFlat = background.get_theme_stylebox("panel").duplicate()

    match current_state:
        CellState.NORMAL:
            style.bg_color = theme_mgr.get_cell_bg()
            letter_label.add_theme_color_override("font_color", theme_mgr.get_text_primary())
        CellState.SELECTED:
            style.bg_color = theme_mgr.get_selecting_color()
            letter_label.add_theme_color_override("font_color", theme_mgr.get_text_primary())
        CellState.FOUND:
            style.bg_color = theme_mgr.get_found_color(found_color_index)
            letter_label.add_theme_color_override("font_color", Color.WHITE)
        CellState.HINT:
            style.bg_color = theme_mgr.get_hint_color()
            letter_label.add_theme_color_override("font_color", Color.WHITE)
        CellState.FALSE_LEAD:
            style.bg_color = theme_mgr.get_cell_bg()  # Normal과 동일

    background.add_theme_stylebox_override("panel", style)

# TV 포커스 처리

    focus_indicator.visible = true

func _on_focus_exited() -> void:
    focus_indicator.visible = false
```

```gdscript
# bottom_tab_bar.gd
extends PanelContainer

signal tab_selected(tab_index: int)

@export var active_color: Color = Color(0.129, 0.588, 0.953)   # #2196F3
@export var inactive_color: Color = Color(0.620, 0.620, 0.620) # #9E9E9E
@export var locked_color: Color = Color(0.741, 0.741, 0.741)   # #BDBDBD

var current_tab: int = 2  # 기본: Home 탭 (중앙)
var tab_items: Array[Control] = []
var tab_locked: Array[bool] = [false, false, false, true, false]  # Collection만 잠금

func _ready() -> void:
    tab_items = [
        $HBoxContainer/TabItem_Daily,
        $HBoxContainer/TabItem_Team,
        $HBoxContainer/TabItem_Home,
        $HBoxContainer/TabItem_Collection,
        $HBoxContainer/TabItem_Shop,
    ]
    _update_lock_states()
    _update_visual()
    for i in range(tab_items.size()):
        tab_items[i].gui_input.connect(_on_tab_input.bind(i))

func _update_lock_states() -> void:
    var player_level: int = SaveManager.get_current_level()
    tab_locked[0] = player_level < 24   # Daily: Lv.24 해금
    tab_locked[1] = player_level < 44   # Team: Lv.44 해금
    tab_locked[3] = true                 # Collection: Coming Soon

func select_tab(index: int) -> void:
    if tab_locked[index]:
        if index == 1:  # Team — 잠금 화면 표시
            tab_selected.emit(index)
        return
    current_tab = index
    _update_visual()
    tab_selected.emit(index)

func set_visible_animated(show: bool) -> void:
    var tween = create_tween()
    if show:
        visible = true
        tween.tween_property(self, "modulate:a", 1.0, 0.2)
    else:
        tween.tween_property(self, "modulate:a", 0.0, 0.2)
        tween.tween_callback(func(): visible = false)

func _update_visual() -> void:
    for i in range(tab_items.size()):
        var color: Color
        if tab_locked[i]:
            color = locked_color
        elif i == current_tab:
            color = active_color
        else:
            color = inactive_color
        tab_items[i].get_node("TabIcon").modulate = color
        tab_items[i].get_node("TabLabel").add_theme_color_override("font_color", color)
        var lock_badge = tab_items[i].get_node_or_null("LockBadge")
        if lock_badge:
            lock_badge.visible = tab_locked[i]

func _on_tab_input(event: InputEvent, index: int) -> void:
    if event is InputEventMouseButton and event.pressed:
        select_tab(index)
```

```gdscript
# toggle_switch.gd
extends Button

signal toggled_custom(is_on: bool)

@onready var track: Panel = $Track
@onready var circle: Panel = $Track/Circle

var is_on: bool = false

func _ready() -> void:
    pressed.connect(_on_pressed)
    _update_visual(false)

func set_on(value: bool, animate: bool = true) -> void:
    is_on = value
    _update_visual(animate)

func _on_pressed() -> void:
    is_on = not is_on
    _update_visual(true)
    toggled_custom.emit(is_on)

func _update_visual(animate: bool) -> void:
    var track_style: StyleBoxFlat = track.get_theme_stylebox("panel").duplicate()
    var target_x: float

    if is_on:
        track_style.bg_color = Color(0.298, 0.686, 0.314)  # #4CAF50
        target_x = track.size.x - circle.size.x - 2.0
    else:
        track_style.bg_color = Color(0.741, 0.741, 0.741)  # #BDBDBD
        target_x = 2.0

    track.add_theme_stylebox_override("panel", track_style)

    if animate:
        var tween = create_tween()
        tween.tween_property(circle, "position:x", target_x, 0.15) \
             .set_ease(Tween.EASE_OUT)
    else:
        circle.position.x = target_x
```

```gdscript
# background_manager.gd
extends Node

# 테마별 배경 이미지 매핑
const THEME_BACKGROUNDS: Dictionary = {
    "animals":    "res://assets/backgrounds/safari_grassland.jpg",
    "food":       "res://assets/backgrounds/rustic_kitchen.jpg",
    "music":      "res://assets/backgrounds/concert_lights.jpg",
    "mythology":  "res://assets/backgrounds/ancient_temple.jpg",
    "ocean":      "res://assets/backgrounds/tropical_beach.jpg",
    "science":    "res://assets/backgrounds/science_lab.jpg",
    "space":      "res://assets/backgrounds/milky_way.jpg",
    "sports":     "res://assets/backgrounds/stadium_field.jpg",
}

# 기본 배경 (테마 미지정 시)
const DEFAULT_BACKGROUND: String = "res://assets/backgrounds/tropical_beach.jpg"

func get_background_path(theme_id: String) -> String:
    return THEME_BACKGROUNDS.get(theme_id.to_lower(), DEFAULT_BACKGROUND)

func apply_background(bg_rect: TextureRect, theme_id: String) -> void:
    var path = get_background_path(theme_id)
    var texture = load(path) as Texture2D
    if texture:
        bg_rect.texture = texture

func transition_background(bg_rect: TextureRect, old_theme: String, new_theme: String) -> void:
    if old_theme == new_theme:
        return
    # 크로스페이드 전환 (1.0초)
    var tween = create_tween()
    tween.tween_property(bg_rect, "modulate:a", 0.0, 0.5)
    tween.tween_callback(func():
        apply_background(bg_rect, new_theme)
    )
    tween.tween_property(bg_rect, "modulate:a", 1.0, 0.5)
```

| 단어 발견 | 셀 스케일 pulse (1.0 -> 1.2 -> 1.0) + 색상 전환 | 0.15s + 0.15s | ease_out | Tween |
| 단어목록 취소선 | 좌 -> 우 선형 확장 | 0.3s | linear | Tween |
| 스테이지 클리어 등급 | 뱃지 bounce (0 -> 1.3 -> 1.0) | 0.5s | ease_out_back | Tween |
| 점수 카운트업 | 0 -> 최종 점수 숫자 증가 | 1.0s | ease_out | Tween |
| 화면 전환 (Push) | slide_left 또는 slide_right | 0.3s | ease_in_out | Tween |
| 화면 전환 (Tab) | crossfade | 0.2s | ease_in_out | Tween |
| 코인 획득 | 코인 아이콘 날아감 + 잔액 bounce | 0.5s + 0.1s | ease_in (fly) + ease_out (bounce) | Tween |
| 힌트 활성화 | 셀 glow 펄스 (alpha 0.5 -> 1.0 반복) | 0.5s/cycle | ease_in_out | Tween (loop) |
| 팝업 진입 | scale 0.8 -> 1.0 + alpha 0 -> 1 | 0.25s | ease_out | Tween |
| 팝업 퇴장 | alpha 1 -> 0 | 0.2s | ease_in | Tween |
| 토스트 표시 | fade_in + 유지 + fade_out | 0.15s + 2.0s + 0.3s | linear | Tween |
| 버튼 터치 피드백 | scale 1.0 -> 0.95 -> 1.0 | 0.08s + 0.08s | ease_out | Tween |
| 격자 셀 등장 | 웨이브 scale 0 -> 1 (셀 간 0.02s 딜레이) | 0.15s/cell | ease_out_back | Tween |
| 클리어 웨이브 | 전체 셀 순차 색상 변경 | 1.5s (전체) | linear | Tween |

### 8.7 Godot Tween vs AnimationPlayer 사용 기준

| 기준 | Tween | AnimationPlayer |
|------|:-----:|:---------------:|
| 단순 속성 변경 (위치, 색상, 크기) | O (권장) | 가능 |
| 복잡한 키프레임 시퀀스 | 가능하나 장황 | O (권장) |
| 런타임 동적 값 기반 | O (권장) | 어려움 |
| 에디터에서 시각적 편집 | 불가 | O (권장) |
| 코드 줄 수 | 적음 | .tscn 분리 필요 |

**본 프로젝트 기준**: 대부분의 UI 애니메이션은 **Tween**으로 구현한다. 런타임에 동적 값(색상, 위치)이 결정되는 경우가 많기 때문이다. 복잡한 시퀀스가 필요한 경우(클리어 웨이브 등)에만 AnimationPlayer를 고려한다.

```gdscript
func calculate_cell_size(grid_width: int, grid_height: int) -> float:
    var viewport_size = get_viewport().get_visible_rect().size
    var available_width = viewport_size.x - GRID_MARGIN * 2  # 좌우 마진
    var available_height = viewport_size.y * 0.5  # 화면 높이의 50% 할당

    var cell_by_width = available_width / grid_width
    var cell_by_height = available_height / grid_height

    var cell_size = min(cell_by_width, cell_by_height)

    # 플랫폼별 최소/최대 제한
    if LayoutManager.is_tv():
        cell_size = clamp(cell_size, 56.0, 80.0)
    else:
        cell_size = clamp(cell_size, 40.0, 60.0)

    return cell_size
```

### 11.4 해상도별 에셋 크기 가이드

Godot의 `canvas_items` Stretch Mode는 단일 해상도 에셋으로 자동 스케일링을 지원한다. 따라서 별도의 @1x/@2x/@3x 에셋 분리는 불필요하다. 다만 다음 기준으로 에셋을 준비한다.

| 에셋 유형 | 기준 크기 | 비고 |
|---------|:---------:|------|
| 아이콘 (UI) | 48x48 px | @2x 기준 (1080px 뷰포트 기준) |
| 아바타 | 512x512 px | 원형 마스크 적용, 충분한 해상도 |
| 랭크 뱃지 | 256x256 px | 결과 화면 대형 표시 |
| 배경 일러스트 | 1080x1920 px | 뷰포트 전체 커버 |
| 앱 아이콘 | 1024x1024 px | 마스터, 각 플랫폼별 자동 축소 |
| 9-patch 패널 | 원본 크기 유지 | stretch_margin으로 확장 |

### 11.5 Godot Import 설정 권장값

```
아이콘/UI 에셋:
  Compress > Mode: Lossless
  Mipmaps: Off (2D UI에 불필요)
  Filter: Linear (기본값)
  Repeat: Disabled

배경 이미지:
  Compress > Mode: VRAM Compressed (ASTC 6x6 for mobile)
  Mipmaps: Off
  Filter: Linear

폰트:
  Subpixel Positioning: Auto
  Antialiasing: LCD (데스크톱), Grayscale (모바일)
```
