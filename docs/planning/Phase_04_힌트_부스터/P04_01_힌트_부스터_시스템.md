## 1. 힌트 시스템 개요

### 1.1 설계 철학

힌트는 플레이어가 **막혔을 때 진행을 도와주는 유료(코인) 보조 수단**이다. 힌트를 남용하면 점수 등급이 낮아지고(C등급), DDA가 난이도를 하향 조정할 수 있으므로 자연스러운 비용-보상 균형이 형성된다.

### 1.2 핵심 원칙

| 원칙 | 설명 |
|------|------|
| **코인 기반** | 모든 힌트는 코인을 소비한다. 무료 힌트는 없다. |
| **단일 활성** | 동시에 하나의 힌트만 활성화할 수 있다 (FullReveal 제외, 즉시 처리). |
| **등급 영향** | 힌트 사용 횟수가 스테이지 클리어 등급에 영향을 미친다 (P01_01 참조). |
| **DDA 연동** | 힌트 사용 횟수가 DDA 지표로 수집된다 (P03_01 참조). |
| **모드 제한** | TimerExtend 힌트는 Time Attack 모드에서만 사용 가능하다. |

### 1.3 HintType 열거형

```gdscript
## hint_type.gd
## 힌트 종류를 정의하는 열거형.
class_name HintType

enum Type {
	FIRST_LETTER,    ## 첫 글자 표시
	FULL_REVEAL,     ## 전체 단어 공개
	MAGNIFIER,       ## 돋보기 (영역 하이라이트)
	TIMER_EXTEND,    ## 타이머 연장 (Time Attack 전용)
	DIRECTION_SHOW,  ## 방향 표시
}
```

## 2. 힌트 종류

### 2.1 첫 글자 표시 (FirstLetter)

| 항목 | 내용 |
|------|------|
| **비용** | 100 코인 |
| **유형** | 기존 (GDD 원본) |
| **대상** | 미발견 단어 중 랜덤 1개 |
| **효과** | 대상 단어의 **첫 번째 셀**을 주황색(#FF8F00)으로 하이라이트 |
| **지속** | 플레이어가 해당 단어를 찾거나 다른 힌트를 사용할 때까지 |
| **점수** | 힌트 단어를 찾아도 정상 점수 부여 (단, 스테이지 등급에 영향) |

**인터랙션 흐름:**

1. 플레이어가 FirstLetter 버튼을 탭한다.
2. 코인이 충분하면 100코인 차감, 미발견 단어 중 랜덤 선택.
3. 선택된 단어의 첫 번째 셀이 주황색으로 변경된다 (CellState: `HintFirstLetter`).
4. 플레이어가 힌트 셀에서 드래그를 시작한다.
5. **성공**: 단어 발견 -> 힌트 상태 해제, 정상 점수 부여.
6. **실패**: 드래그 취소/오답 -> 힌트 셀이 다시 `HintFirstLetter` 상태로 복원 (추가 비용 없음).

```gdscript
## 첫 글자 힌트 활성화
func _activate_first_letter() -> void:
	var unfound: Array = grid_data.placed_words.filter(
		func(pw: PlacedWord) -> bool: return not pw.is_found
	)
	if unfound.is_empty():
		return

	# 랜덤 단어 선택
	active_hint_word = unfound[randi() % unfound.size()]
	active_hint_type = HintType.Type.FIRST_LETTER

	# 첫 번째 셀 좌표
	var first_cell: Vector2i = active_hint_word.start_pos
	hint_activated.emit(HintType.Type.FIRST_LETTER, active_hint_word, [first_cell])
```

### 2.2 전체 단어 공개 (FullReveal)

| 항목 | 내용 |
|------|------|
| **비용** | 200 코인 |
| **유형** | **신규** (시장 조사 기반 추가) |
| **대상** | 미발견 단어 중 랜덤 1개 |
| **효과** | 대상 단어의 **모든 셀**을 하이라이트하고, 즉시 **발견 상태로 처리** |
| **지속** | 즉시 처리 (활성 힌트 슬롯을 차지하지 않음) |
| **점수** | 해당 단어에 대한 **점수 없음** (0점). 스테이지 클리어 보너스에만 영향. |

**인터랙션 흐름:**

1. 플레이어가 FullReveal 버튼을 탭한다.
2. 코인이 충분하면 200코인 차감, 미발견 단어 중 랜덤 선택.
3. 선택된 단어의 모든 셀이 발견 색상으로 변경되고, `is_found = true` 처리.
4. WordBank에서 해당 단어에 취소선 표시.
5. 힌트 활성 상태는 즉시 해제된다 (다른 힌트와 병행 가능).

```gdscript
## 전체 단어 공개 힌트 활성화
func _activate_full_reveal() -> void:
	var unfound: Array = grid_data.placed_words.filter(
		func(pw: PlacedWord) -> bool: return not pw.is_found
	)
	if unfound.is_empty():
		return

	var target: PlacedWord = unfound[randi() % unfound.size()]

	# 즉시 발견 처리
	target.is_found = true

	# 모든 셀 좌표 전달
	hint_activated.emit(HintType.Type.FULL_REVEAL, target, target.cells.duplicate())

	# 즉시 해제 (활성 힌트 슬롯 차지 안 함)
	hint_cleared.emit()
```

### 2.3 돋보기 (Magnifier)

| 항목 | 내용 |
|------|------|
| **비용** | 150 코인 |
| **유형** | **신규** (시장 조사 기반 추가) |
| **대상** | 미발견 단어 중 랜덤 1개 |
| **효과** | 대상 단어의 **중심점 기준 3x3 영역**에 은은한 글로우(glow) 표시 |
| **지속** | **5초** 후 자동 해제 |
| **점수** | 해당 단어를 직접 찾으면 정상 점수 부여 |

**인터랙션 흐름:**

1. 플레이어가 Magnifier 버튼을 탭한다.
2. 코인이 충분하면 150코인 차감, 미발견 단어 중 랜덤 선택.
3. 대상 단어의 중심 셀을 기준으로 3x3 영역(최대 9셀)에 글로우 오버레이 표시.
4. 5초 타이머 시작.
5. 5초 경과 또는 플레이어가 해당 단어를 찾으면 글로우 해제.
6. 플레이어는 여전히 직접 단어를 찾아야 한다.

```gdscript
## 돋보기 힌트 활성화
## 단어의 중심 셀 기준 3x3 영역을 하이라이트한다.
const MAGNIFIER_RADIUS: int = 1  ## 3x3 = 반경 1
const MAGNIFIER_DURATION: float = 5.0  ## 글로우 지속 시간 (초)

func _activate_magnifier() -> void:
	var unfound: Array = grid_data.placed_words.filter(
		func(pw: PlacedWord) -> bool: return not pw.is_found
	)
	if unfound.is_empty():
		return

	active_hint_word = unfound[randi() % unfound.size()]
	active_hint_type = HintType.Type.MAGNIFIER

	# 단어의 중심 셀 계산
	var word_cells: Array = active_hint_word.cells
	var center_index: int = word_cells.size() / 2
	var center_cell: Vector2i = word_cells[center_index]

	# 3x3 영역 셀 수집 (격자 범위 내만)
	var area_cells: Array[Vector2i] = []
	for dy in range(-MAGNIFIER_RADIUS, MAGNIFIER_RADIUS + 1):
		for dx in range(-MAGNIFIER_RADIUS, MAGNIFIER_RADIUS + 1):
			var pos := Vector2i(center_cell.x + dx, center_cell.y + dy)
			if grid_data.is_in_bounds(pos):
				area_cells.append(pos)

	hint_activated.emit(HintType.Type.MAGNIFIER, active_hint_word, area_cells)

	# 5초 후 자동 해제 (호출 측에서 Timer 또는 Tween으로 처리)
	_start_magnifier_timer()


func _start_magnifier_timer() -> void:
	# SceneTree timer를 사용하여 5초 후 해제
	# 실제 구현에서는 GameScreen이 Timer 노드를 관리한다.
	await Engine.get_main_loop().create_timer(MAGNIFIER_DURATION).timeout
	if active_hint_type == HintType.Type.MAGNIFIER:
		_clear_active_hint()
```

### 2.4 타이머 연장 (TimerExtend)

| 항목 | 내용 |
|------|------|
| **비용** | 80 코인 |
| **유형** | **신규** (시장 조사 기반 추가) |
| **대상** | 해당 스테이지의 타이머 |
| **효과** | 남은 시간에 **+30초** 추가 |
| **가용성** | **Time Attack 모드에서만** 사용 가능 (다른 모드에서는 버튼 자체가 숨겨짐) |
| **점수** | 점수에 직접 영향 없음 (단, 힌트 사용 횟수로 카운트됨) |

**인터랙션 흐름:**

1. Time Attack 모드에서 플레이어가 TimerExtend 버튼을 탭한다.
2. 코인이 충분하면 80코인 차감.
3. 타이머에 +30초가 즉시 추가된다.
4. 힌트 사용 횟수 1회 증가 (DDA 및 등급 판정에 영향).
5. 활성 힌트 슬롯을 차지하지 않는다 (다른 힌트와 병행 가능).

```gdscript
## 타이머 연장 힌트
const TIMER_EXTEND_SECONDS: float = 30.0

signal timer_extended(seconds: float)

func _activate_timer_extend() -> void:
	# 즉시 처리, 활성 힌트 슬롯 차지 안 함
	timer_extended.emit(TIMER_EXTEND_SECONDS)
	hint_cleared.emit()
```

### 2.5 방향 표시 (DirectionShow)

| 항목 | 내용 |
|------|------|
| **비용** | 120 코인 |
| **유형** | **신규** (시장 조사 기반 추가) |
| **대상** | 미발견 단어 중 랜덤 1개 |
| **효과** | 대상 단어의 **첫 번째 셀**에 방향 화살표 아이콘 표시 |
| **지속** | 해당 단어를 찾거나 다른 힌트를 사용할 때까지 |
| **점수** | 해당 단어를 직접 찾으면 정상 점수 부여 |

**인터랙션 흐름:**

1. 플레이어가 DirectionShow 버튼을 탭한다.
2. 코인이 충분하면 120코인 차감, 미발견 단어 중 랜덤 선택.
3. 선택된 단어의 첫 번째 셀 위에 **작은 화살표 아이콘**이 표시된다.
4. 화살표 방향은 단어의 배치 방향(`direction` 벡터)과 일치한다.
5. 플레이어가 해당 단어를 찾으면 화살표가 사라진다.
6. 다른 힌트를 사용하면 기존 방향 힌트가 해제된다.

```gdscript
## 방향 표시 힌트 활성화
func _activate_direction_show() -> void:
	var unfound: Array = grid_data.placed_words.filter(
		func(pw: PlacedWord) -> bool: return not pw.is_found
	)
	if unfound.is_empty():
		return

	active_hint_word = unfound[randi() % unfound.size()]
	active_hint_type = HintType.Type.DIRECTION_SHOW

	# 첫 번째 셀 + 방향 정보를 함께 전달
	var first_cell: Vector2i = active_hint_word.start_pos
	hint_activated.emit(HintType.Type.DIRECTION_SHOW, active_hint_word, [first_cell])
	# UI 측에서 active_hint_word.direction을 읽어 화살표 방향을 결정한다
```

**방향 벡터와 화살표 매핑:**

| Direction Vector | 화살표 방향 | 아이콘 회전 (도) |
|------------------|------------|-----------------|
| `(1, 0)` | -> | 0 |
| `(-1, 0)` | <- | 180 |
| `(0, 1)` | v | 90 |
| `(0, -1)` | ^ | 270 |
| `(1, 1)` | \\ (우하) | 45 |
| `(-1, -1)` | \\ (좌상) | 225 |
| `(1, -1)` | / (우상) | 315 |
| `(-1, 1)` | / (좌하) | 135 |

```gdscript
## 방향 벡터를 화살표 회전 각도(도)로 변환한다.
static func direction_to_rotation(dir: Vector2i) -> float:
	return rad_to_deg(atan2(float(dir.y), float(dir.x)))
```

## 3. 부스터 시스템

부스터는 힌트와 달리 **직접적으로 정답을 알려주지 않고**, 플레이 환경을 변경하여 간접적으로 도움을 주는 기능이다.

### 3.1 셔플 (Shuffle)

| 항목 | 내용 |
|------|------|
| **비용** | **무료** |
| **사용 횟수** | 스테이지당 **1회** |
| **효과** | 빈 칸(미사용 셀)의 문자만 셔플. 배치된 단어의 글자는 이동하지 않음. |
| **시각 효과** | 셀들이 잠시 흔들리다가 새 위치로 정렬되는 애니메이션 |

> **설계 의도:** 셔플은 "시야를 리프레시"하는 용도이다. 오래 응시하다 놓친 패턴을 새로운 배치에서 발견할 수 있도록 돕는다. 실제로 단어 위치는 변하지 않으므로 게임 난이도에 직접적 영향은 없다. 빈 칸 문자의 시각적 변화가 플레이어에게 "새로운 시각"을 제공한다.

```gdscript
## shuffle_booster.gd
## 빈 칸의 문자만 셔플하는 부스터.
## 배치된 단어의 글자 위치는 절대 변경하지 않는다.

## 빈 칸 셔플을 수행한다.
## 배치된 단어가 차지하는 셀을 제외한 나머지 셀의 문자를 무작위로 재배치한다.
func shuffle_empty_cells(grid_data: GridData) -> void:
	# 1. 배치된 단어가 차지하는 셀 좌표 집합 생성
	var word_cells: Dictionary = {}  # Vector2i -> true
	for pw in grid_data.placed_words:
		for cell_pos in pw.cells:
			word_cells[cell_pos] = true

	# 2. 빈 칸(단어가 아닌 셀)의 문자 수집
	var empty_chars: Array[String] = []
	var empty_positions: Array[Vector2i] = []

	for y in range(grid_data.height):
		for x in range(grid_data.width):
			var pos := Vector2i(x, y)
			if not word_cells.has(pos):
				empty_chars.append(grid_data.grid[y][x])
				empty_positions.append(pos)

	# 3. 문자 배열 셔플
	empty_chars.shuffle()

	# 4. 셔플된 문자를 원래 빈 칸 위치에 재배치
	for i in range(empty_positions.size()):
		var pos: Vector2i = empty_positions[i]
		grid_data.grid[pos.y][pos.x] = empty_chars[i]
```

| 유형 | 이름 | 비용 (코인) | 가용 모드 | 효과 지속 | 활성 슬롯 점유 | 점수 영향 |

| 힌트 | 방향 표시 (DirectionShow) | 120 | 전체 | 단어 발견 시까지 | O | 정상 점수 |
| 힌트 | 전체 단어 공개 (FullReveal) | 200 | 전체 | 즉시 | X (즉시 처리) | **0점** (해당 단어) |
| 힌트 | 타이머 연장 (TimerExtend) | 80 | **Time Attack만** | 즉시 | X (즉시 처리) | 간접 (시간 보너스 변동) |
| 부스터 | 셔플 (Shuffle) | **무료** | 전체 | 즉시 | X | 없음 |

### 코인 수급 참고

| 획득 방법 | 코인량 | 비고 |
|-----------|--------|------|
| 단어 발견 | 10~20 | 단어 길이에 따라 변동 |
| 스테이지 클리어 | 50~150 | 등급에 따라 변동 |
| 광고 시청 | 100~200 | Rewarded Ad |
| Daily Challenge | 100 | 1일 1회 |

> **밸런스 의도:** 1 스테이지에서 약 100~250코인을 획득할 수 있으므로, FirstLetter 힌트(100코인)는 1 스테이지 수입의 약 절반에 해당한다. FullReveal(200코인)은 약 1 스테이지 수입 전체에 해당하여 신중하게 사용하도록 유도한다.

## 7. hint_manager.gd 설계

```gdscript
## hint_manager.gd
## 힌트 시스템을 관리하는 매니저 클래스.
## 코인 소비, 힌트 활성화/해제, 상태 추적을 담당한다.
## C# 원본: HintManager.cs (v5.0 FirstLetter만 -> v6.0 확장)
class_name HintManager
extends RefCounted


## --- Signals ---

## 힌트가 활성화되었을 때 발신한다.
## type: HintType.Type, word: 대상 PlacedWord, cells: 하이라이트할 셀 좌표 배열
signal hint_activated(type: int, word: PlacedWord, cells: Array[Vector2i])

## 활성 힌트가 해제되었을 때 발신한다.
signal hint_cleared()

## 코인이 부족하여 힌트를 사용할 수 없을 때 발신한다.
signal insufficient_coins()

## 타이머 연장 힌트 사용 시 발신한다.
signal timer_extended(seconds: float)


## --- 힌트 비용 상수 ---

const COST_FIRST_LETTER: int = 100
const COST_MAGNIFIER: int = 150
const COST_DIRECTION_SHOW: int = 120
const COST_FULL_REVEAL: int = 200
const COST_TIMER_EXTEND: int = 80

## 돋보기 글로우 지속 시간 (초)
const MAGNIFIER_DURATION: float = 5.0

## 타이머 연장 시간 (초)
const TIMER_EXTEND_SECONDS: float = 30.0

## 돋보기 영역 반경 (3x3 = 반경 1)
const MAGNIFIER_RADIUS: int = 1


## --- 상태 ---

## 현재 활성화된 힌트 유형 (-1: 없음)
var active_hint_type: int = -1

## 현재 활성화된 힌트의 대상 단어
var active_hint_word: PlacedWord = null

## 격자 데이터 참조
var grid_data: GridData = null

## 이 스테이지에서 사용한 총 힌트 횟수
var total_hints_used: int = 0

## 셔플 남은 횟수
var shuffle_remaining: int = 1


## 초기화.
## 새 스테이지 시작 시 호출한다.
func setup(p_grid_data: GridData) -> void:
	grid_data = p_grid_data
	active_hint_type = -1
	active_hint_word = null
	total_hints_used = 0
	shuffle_remaining = 1


## 힌트를 사용한다.
## 코인이 부족하면 insufficient_coins 신호를 발신하고 false를 반환한다.
## 성공 시 true를 반환한다.
func use_hint(type: int) -> bool:
	var cost := get_cost(type)

	# 코인 잔액 확인
	if not CoinManager.can_spend(cost):
		insufficient_coins.emit()
		return false

	# 기존 활성 힌트 해제 (FullReveal, TimerExtend는 즉시 처리이므로 예외)
	if type != HintType.Type.FULL_REVEAL and type != HintType.Type.TIMER_EXTEND:
		if active_hint_type >= 0:
			_clear_active_hint()

	# 코인 차감
	CoinManager.spend(cost)
	total_hints_used += 1

	# 유형별 활성화
	match type:
		HintType.Type.FIRST_LETTER:
			_activate_first_letter()
		HintType.Type.FULL_REVEAL:
			_activate_full_reveal()
		HintType.Type.MAGNIFIER:
			_activate_magnifier()
		HintType.Type.TIMER_EXTEND:
			_activate_timer_extend()
		HintType.Type.DIRECTION_SHOW:
			_activate_direction_show()

	return true


## 셔플 부스터를 사용한다.
## 무료이므로 코인 차감 없이 사용 횟수만 감소시킨다.
func use_shuffle() -> bool:
	if shuffle_remaining <= 0:
		return false

	shuffle_remaining -= 1
	# 실제 셔플 로직은 호출 측(GameScreen)에서 처리
	return true


## 힌트 비용을 반환한다.
static func get_cost(type: int) -> int:
	match type:
		HintType.Type.FIRST_LETTER:
			return COST_FIRST_LETTER
		HintType.Type.FULL_REVEAL:
			return COST_FULL_REVEAL
		HintType.Type.MAGNIFIER:
			return COST_MAGNIFIER
		HintType.Type.TIMER_EXTEND:
			return COST_TIMER_EXTEND
		HintType.Type.DIRECTION_SHOW:
			return COST_DIRECTION_SHOW
		_:
			push_warning("HintManager: 알 수 없는 힌트 유형 %d" % type)
			return 0


## 활성 힌트가 있는지 확인한다.
func has_active_hint() -> bool:
	return active_hint_type >= 0


## 활성 힌트의 대상 셀을 반환한다 (FirstLetter 용).
func get_hint_cell() -> Vector2i:
	if active_hint_word != null:
		return active_hint_word.start_pos
	return Vector2i(-1, -1)


## 힌트 셀에서 드래그가 성공했을 때 호출한다.
## FirstLetter 힌트의 대상 단어를 찾았을 때 사용한다.
func on_hint_drag_succeeded() -> void:
	_clear_active_hint()


## 힌트 셀에서 드래그가 실패했을 때 호출한다.
## 힌트 상태를 유지하여 재시도를 허용한다.
func on_hint_drag_failed() -> void:
	# 힌트 상태 유지 (추가 비용 없음)
	pass


## 이 스테이지에서 사용한 힌트 총 횟수를 반환한다.
func get_total_used() -> int:
	return total_hints_used


## --- 내부 함수 ---

func _activate_first_letter() -> void:
	var unfound: Array = grid_data.placed_words.filter(
		func(pw: PlacedWord) -> bool: return not pw.is_found
	)
	if unfound.is_empty():
		return

	active_hint_word = unfound[randi() % unfound.size()]
	active_hint_type = HintType.Type.FIRST_LETTER

	var first_cell: Vector2i = active_hint_word.start_pos
	hint_activated.emit(HintType.Type.FIRST_LETTER, active_hint_word, [first_cell])


func _activate_full_reveal() -> void:
	var unfound: Array = grid_data.placed_words.filter(
		func(pw: PlacedWord) -> bool: return not pw.is_found
	)
	if unfound.is_empty():
		return

	var target: PlacedWord = unfound[randi() % unfound.size()]
	target.is_found = true

	hint_activated.emit(HintType.Type.FULL_REVEAL, target, target.cells.duplicate())
	hint_cleared.emit()


func _activate_magnifier() -> void:
	var unfound: Array = grid_data.placed_words.filter(
		func(pw: PlacedWord) -> bool: return not pw.is_found
	)
	if unfound.is_empty():
		return

	active_hint_word = unfound[randi() % unfound.size()]
	active_hint_type = HintType.Type.MAGNIFIER

	# 중심 셀 계산
	var word_cells: Array = active_hint_word.cells
	var center_index: int = word_cells.size() / 2
	var center_cell: Vector2i = word_cells[center_index]

	# 3x3 영역 수집
	var area_cells: Array[Vector2i] = []
	for dy in range(-MAGNIFIER_RADIUS, MAGNIFIER_RADIUS + 1):
		for dx in range(-MAGNIFIER_RADIUS, MAGNIFIER_RADIUS + 1):
			var pos := Vector2i(center_cell.x + dx, center_cell.y + dy)
			if grid_data.is_in_bounds(pos):
				area_cells.append(pos)

	hint_activated.emit(HintType.Type.MAGNIFIER, active_hint_word, area_cells)


func _activate_timer_extend() -> void:
	timer_extended.emit(TIMER_EXTEND_SECONDS)
	hint_cleared.emit()


func _activate_direction_show() -> void:
	var unfound: Array = grid_data.placed_words.filter(
		func(pw: PlacedWord) -> bool: return not pw.is_found
	)
	if unfound.is_empty():
		return

	active_hint_word = unfound[randi() % unfound.size()]
	active_hint_type = HintType.Type.DIRECTION_SHOW

	var first_cell: Vector2i = active_hint_word.start_pos
	hint_activated.emit(HintType.Type.DIRECTION_SHOW, active_hint_word, [first_cell])


func _clear_active_hint() -> void:
	active_hint_type = -1
	active_hint_word = null
	hint_cleared.emit()
```
