## 1. 시장 현황 & 타겟 유저

### 1.1 시장 규모

- 퍼즐 게임은 전체 모바일 게임 다운로드의 **28%** 차지 (2024 기준)
- 전 세계 **12억 명** 이상이 퍼즐 게임을 정기적으로 플레이
- 워드/퍼즐 게임은 캐주얼 게임 중 가장 높은 DAU(일일 활성 유저) 보유

### 1.2 핵심 타겟 유저 프로파일

| 항목 | 데이터 |
|------|--------|
| **주 성별** | 여성 75% |
| **핵심 연령대** | 35~49세 피크, 25~54세 전체의 60% |
| **플레이 동기** | 일상 스트레스 해소, "나만의 시간(me time)" |
| **평균 세션 시간** | 여성 19.1분/일, 남성 17.6분/일 |
| **플레이 환경** | 통근, 취침 전, 점심시간 등 짧은 여가 |

### 1.3 핵심 UX 감성

> **"릴렉싱(Relaxing)" + "성취감(Achievement)"**
> 경쟁이 아닌 자기 페이스로 즐기는 **마음 안정형 게임**이 압도적 다수

## 2. 게임별 카테고리 분석

### 카테고리 A: 자연/풍경 테마 — 힐링 감성형

#### Wordscapes (PeopleFun) ★ 가장 중요한 레퍼런스
- **비주얼 스타일**: 사진급 자연 배경 + 미니멀 크로스워드 UI. 배경이 주인공
- **컬러 팔레트**: 쪽빛 하늘, 에메랄드 숲, 노을 오렌지, 황금빛 일몰
- **UI 테마**: Forest / Canyon / Passage / Beach / Desert / Mountain 등 구역별 테마
- **배경**: 실제 자연 사진에 가까운 고퀄리티 일러스트 또는 사진 블렌드
- **타일**: 흰색 원형 글자판 + 연한 테두리. 선택 시 색상 변화. 그림자 없는 플랫 스타일
- **애니메이션**: 단어 완성 시 타일이 빛나며 사라짐. 레벨 클리어 시 나뭇잎/꽃잎 파티클
- **BGM**: 부드러운 피아노 + 자연 앰비언트 레이어링 (새소리, 바람, 물소리)
- **효과음**: 파도 소리에 가까운 릴렉싱 앰비언트. 단어 완성 시 기분 좋은 차임
- **UX 감성**: 명상적(Meditative). 스트레스 제로. 시간 제한 없음

#### Word Relax (TG Media)
- **비주얼**: Wordscapes와 유사한 자연 풍경 배경. 한층 더 부드러운 색감
- **배경**: 실사에 가까운 자연 풍경 (세계 각지 명소 느낌). 레벨 클리어 시 새 풍경 언락
- **특징**: "여행하는 느낌"을 강조. 20,000+ 레벨
- **UX**: 타이머 없음, 압박감 없음. "완전한 힐링"에 집중

#### Word Beach (Bitmango)
- **테마**: 해변/열대 리조트
- **컬러**: 청록색(Turquoise), 산호색(Coral), 흰 모래 베이지, 야자수 초록
- **배경**: 따뜻하고 밝은 여름 일러스트. 물결치는 애니메이션
- **감성**: 여름 휴가지 분위기. 가볍고 경쾌

#### Word Tiles: Relax n Refresh (Bitmango)
- **배경**: 고요한 자연 풍경 (산, 호수, 숲)
- **BGM**: "Gentle music" — 부드러운 음악 명시
- **시각 효과**: "Gorgeous visual effects" — 타일 처리 시 반짝이는 빛 파티클

---

### 카테고리 B: 음식/캐릭터 테마 — 귀엽고 발랄한 캐주얼형

#### Word Cookies! (BitMango)
- **비주얼**: 쿠키/제과 테마. 따뜻한 갈색, 크림색, 파스텔 핑크
- **배경**: 제과점 느낌의 일러스트. 쿠키 반죽판 모티프
- **타일**: 쿠키 모양의 알파벳 타일. 둥글둥글하고 귀여운 형태
- **애니메이션**: 단어 완성 시 쿠키들이 반짝이며 파티클 튀김
- **BGM**: "Happy background music" — 경쾌하고 달콤한 느낌
- **타겟**: 여성 25~45세. 가볍고 즐거운 무드

#### Alphabear (Spry Fox) ★ 수상작 레퍼런스
- **비주얼**: 귀여운 네모난 곰 캐릭터 중심. 픽셀 아트 + 카툰 혼합
- **배경**: 동화 같은 목가적 배경. 캐릭터가 격자판을 채우며 성장
- **컬러**: 따뜻한 파스텔톤. 브라운, 베이지, 소프트 그린, 크림
- **애니메이션**: 곰 캐릭터가 레벨이 올라갈수록 커짐
- **효과음**: 유쾌하고 장난스러운 효과음. 캐릭터 특화 사운드
- **수상**: Google Play Award "Best Standout Indie" 2016, Gamezebo 올해의 게임 2015

#### Word Farm Adventure (Mad Brain Games)
- **비주얼**: 농장 + 단어 퍼즐 결합. 선명하고 생동감 있는 색감
- **배경**: 디테일한 농장 일러스트. 동물 캐릭터들
- **특징**: 퍼즐 + 농장 꾸미기 하이브리드. 스토리 드리븐

---

### 카테고리 C: 세계 여행/문화 테마 — 지식과 감성의 결합

#### Words of Wonders (Fugo Games)
- **비주얼**: 세계 7대 불가사의, 에펠탑, 피라미드 등 랜드마크 배경
- **컬러**: 국가/지역별 특색 반영. 이집트=사막색+골드, 프랑스=블루+화이트
- **배경**: 아이코닉한 건축물 일러스트. 상세하고 아름다운 배경 아트
- **UX**: 교육적이면서 릴렉싱

#### CodyCross (Fanatee)
- **비주얼**: 외계인 Cody 캐릭터 중심. Under the Sea, Ancient Egypt, Wild West 등 테마 월드
- **배경**: 테마별로 완전히 다른 비주얼 세계관. 화려하고 세밀한 일러스트
- **특징**: 세계관 몰입형. 탐험 스토리텔링 구조

---

### 카테고리 D: 소셜/경쟁형 — 역동적이고 에너제틱한 디자인

#### Words With Friends (Zynga)
- **비주얼**: 스크래블 보드게임 디지털 버전. 클래식 보드 질감 + 모던 UI
- **컬러**: 초록 보드, 우드 질감, 블루/화이트 포인트 컬러
- **효과음**: 타일 배치 시 나무 블록 소리. 경쟁적 긴장감 BGM
- **특징**: 타일 스타일 커스터마이징 가능

#### Wordament (Microsoft)
- **비주얼**: 클린하고 심플. Microsoft 스타일의 모던 플랫 디자인
- **색상**: 파란색/흰색 Microsoft 팔레트
- **특징**: 실시간 전 세계 동시 대결

---

### 카테고리 E: 미니멀리스트 인디 — 심플하고 엘레강트한 디자인

#### SpellTower (Zach Gage)
- **비주얼**: 극도로 미니멀. 흰색/검정 베이스. 타이포그래피 중심
- **폰트**: 클래식하고 엘레강트한 폰트. 디지털 크로스워드의 감성
- **배경**: 그라디언트 또는 단색. 장식 없음
- **감성**: 지적이고 집중된 경험

#### Letterpress (Loren Brichter) ★ 미니멀 디자인의 교과서
- **비주얼**: "Gorgeous example of minimalist design"
- **특징**: "Barren interface", "natural-feeling animations", "aesthetically pleasing"
- **색상 메커니즘**: 영역 점령 시 플레이어 색상이 채워짐 (파랑 vs 빨강)
- **평가**: 업계에서 미니멀 모바일 디자인의 교과서로 평가

## 3. 비주얼 스타일 트렌드 종합

### 3.1 주류 스타일 분포

| 스타일 | 비율 | 대표 게임 |
|--------|------|----------|
| 자연 배경 힐링형 | 35% | Wordscapes, Word Relax, Word Beach |
| 음식/캐릭터 카툰형 | 25% | Word Cookies, Alphabear, Word Farm |
| 소셜/경쟁 클래식형 | 15% | Words With Friends, Boggle, Wordament |
| 세계 여행/문화형 | 15% | Words of Wonders, CodyCross, Word Trip |
| 미니멀리스트 인디형 | 10% | SpellTower, TypeShift, Letterpress |

### 3.2 공통 비주얼 원칙 5가지

1. **가독성 최우선**: 글자는 언제나 명확하게. 배경이 텍스트를 방해해서는 안 됨
2. **몰입형 배경**: UI 요소를 최소화하고 배경 아트웍으로 감성 전달
3. **레벨 진행에 따른 시각적 변화**: 배경, 색조가 조금씩 변해 신선함 유지
4. **소프트하고 둥근 형태**: 각진 요소보다 원형, 둥근 모서리가 대세
5. **파스텔 & 자연색 우세**: 강렬한 원색보다 부드러운 자연색 계열이 지배적

## 4. 컬러 팔레트 심층 분석

### 4.1 자연/힐링형 컬러 팔레트 (가장 보편적)

```
배경(Sky):    #4A90D9 → #87CEEB  (하늘색 그라디언트)
배경(Forest): #2D6A4F → #52B788  (짙은 초록 → 밝은 초록)
배경(Sunset): #FF6B35 → #FFB347  (오렌지 → 골든)
배경(Night):  #1A1A2E → #16213E → #0F3460  (딥 네이비)
강조(Gold):   #FFD700, #FFC107   (성취/코인)
타일(기본):   #FFFFFF, #F5F5F5   (흰색 계열)
타일(선택):   #FFD54F, #FFCA28   (골드 옐로우)
타일(정답):   #A5D6A7, #4CAF50   (파스텔 그린 → 그린)
```

| Ocean | `#0077B6` (딥 블루) | `#48CAE4` (시안 블루) | `#90E0EF` (라이트 시안) |

### 4.3 공통 UI 색상 공식 (황금 공식)

| 역할 | 색상 코드 | 용도 |
|------|-----------|------|
| 타일 기본 | `#FFFFFF` | 기본 상태 타일 |
| 타일 선택 중 | `#FFD54F` | 드래그 중 강조 |
| 타일 정답 | `#A5D6A7` → `#4CAF50` | 정답 확정 |
| 이미 찾은 단어 | `#BDBDBD` | 완료된 단어 (회색) |
| 코인/보상 | `#FFD700` | 보상, 특별 요소 |
| 힌트/아이템 | `#4FC3F7` | 도움, 신뢰 |
| 오류/실패 | `#FF7675` | 소프트 코랄 (강한 빨강 금지!) |
| 글자 색상 | `#3E2723` | 딥 브라운 — 모든 배경 대비 |
| 텍스트 일반 | `#212121` | 본문 텍스트 |
| 텍스트 보조 | `#757575` | 서브 텍스트 |

## 6. 배경 이미지 스타일 분석

### 6.1 스타일 유형 비교

| 유형 | 대표 게임 | 특징 | 제작 난이도 |
|------|-----------|------|------------|
| **포토리얼 배경** | Wordscapes | 실사 사진에 가까운 고퀄 렌더 | 높음 |
| **스타일라이즈드 일러스트** | CodyCross, Word Cookies | 벡터 기반 선명한 일러스트 | 중간 |
| **그라디언트 배경** | 다수 게임 기본 | 두 색상 그라디언트 | 낮음 |
| **미니멀/단색** | SpellTower, Letterpress | 단색 배경 | 매우 낮음 |

### 6.2 배경 구성 3층 공식

```
전경 (Front):  UI 요소 (버튼, 타일, 점수판)
중경 (Mid):    테마 오브젝트 (나무, 동물, 건물 — Blur 처리)
원경 (Back):   하늘/대기 그라디언트 (가장 단순하게)
```

### 6.3 핵심 배경 처리 원칙

- **Blur 처리**: 배경 중경은 Gaussian Blur 적용 → 타일 가독성 확보
- **명도 조정**: 배경 전체 밝기를 UI보다 30~40% 낮춤 → 충돌 방지
- **채도 조정**: 배경 채도를 약간 낮춰 UI 색상이 더 선명하게 보이도록
- **시차 효과(Parallax)**: 기기 기울기에 따라 배경이 살짝 움직이면 고급감 상승

## 7. 글자/타일 디자인 패턴

### 7.1 타일 형태 분류

| 형태 | 대표 게임 | 특징 |

| **스크래블 타일(사각)** | Words With Friends, Boggle | 나무 질감, 클래식 보드게임 감성 |
| **헥사곤** | Wordscapes Shapes | 벌집 모양. 독특한 레이아웃 |
| **투명/반투명** | 일부 미니멀 게임 | 배경이 비쳐 보임. 모던한 느낌 |

### 7.2 타일 상태별 시각 처리

| 상태 | 색상 | 기타 효과 | 전환 시간 |
|------|------|---------|---------|
| **기본(Idle)** | `#FFFFFF` | 얇은 그림자 | — |
| **선택됨(Selected)** | `#FFD54F` (골드) | 밝기 증가 | 즉시 |
| **드래그 중** | `#FFD54F` + 연결선 표시 | 컬러 라인 | 즉시 |
| **정답 확정** | `#A5D6A7` → `#4CAF50` | 글로우 효과 + 파티클 | 0.2초 |
| **완료(Cleared)** | `#BDBDBD` (그레이) | 채도 제거 | 0.3초 |

## 9. 사운드 디자인 심층 분석

### 9.1 BGM 장르 & 구성

#### 힐링/자연형 BGM (압도적 주류 — 우리 프로젝트 방향)

| 항목 | 사양 |
|------|------|
| **BPM** | 60~85 (느리고 편안하게) |
| **장르** | 앰비언트(Ambient) / Lo-Fi / 뉴에이지(New Age) |
| **주요 악기** | 피아노, 어쿠스틱 기타, 패드 신스, 첼레스타, 마림바 |
| **자연음 레이어** | 새소리, 바람, 물소리, 파도소리 (테마별 선택) |
| **루프 길이** | 2~3분 (반복해도 지루하지 않도록 변주 포함) |
| **키** | C Major 또는 G Major (밝고 따뜻한) |

#### 경쾌/캐주얼형 BGM

| 항목 | 사양 |
|------|------|
| **BPM** | 90~120 |
| **장르** | 팝, 경쾌한 어쿠스틱, 인디 팝 |
| **주요 악기** | 업라이트 피아노, 우쿨렐레, 클라리넷, 글로켄슈필 |
| **분위기** | "Happy background music" — 달콤하고 경쾌 |

#### 소셜/경쟁형 BGM

| 항목 | 사양 |
|------|------|
| **BPM** | 85~110 |
| **장르** | 재즈, 경쾌한 팝, 전자음악 혼합 |
| **분위기** | 집중감 있지만 스트레스 없는 세련된 배경음 |

### 9.2 BGM 레이어 구조

```
Layer 1 (Base):    드론/패드 신스    — 배경 분위기 유지 (항상 존재)
Layer 2 (Melody):  피아노/첼레스타   — 주 멜로디
Layer 3 (Rhythm):  가벼운 플럭 기타 또는 마림바
Layer 4 (Ambient): 환경음           — 테마별 자연음/공간음
```

> 게임 시작 시 Layer 1부터 순차적으로 추가 → 자연스러운 음악 빌드업
> 레벨 클리어 직전 BGM이 약간 고조 → 성취감 극대화

### 9.3 테마별 BGM 환경음 (Layer 4)

| 테마 | 환경음 |
|------|--------|
| Animals | 새소리, 바람, 풀벌레 소리 |
| Ocean | 파도, 갈매기, 물 흐르는 소리 |
| Space | 우주 앰비언트, 딥 신스 드론 |
| Food | 조용한 카페/주방 소리 |
| Music | 피아노 선율 강화, 에코 홀 잔향 |
| Mythology | 신비로운 합창, 고대 악기 드론 |
| Science | 전자음, 버블 소리, 미래 패드 |
| Sports | 경기장 잔향, 경쾌한 비트 |

## 10. 성공/실패 효과음 패턴

### 10.1 단어 발견 성공음

| 단어 유형 | 길이 | 음정 패턴 | 음색 |
|----------|------|----------|------|
| **짧은 단어** (3~4글자) | 0.3~0.5초 | C-E-G 상승 (1~2음) | 피아노 or 첼레스타 + 글리터 |
| **긴 단어** (5~7글자) | 0.6~1.0초 | C-E-G-C 상승 (3~4음) | 피아노 + 하프 글리산도 + 심볼즈 |
| **보너스 단어** (8글자+) | 1.0~1.5초 | 미니 팡파레 | 트럼펫 스타카토 + 코인음 + 파티클 |

### 10.2 타일 클릭/드래그 효과음

| 동작 | 효과음 | 느낌 |
|------|--------|------|
| 타일 선택 시작 | 부드러운 클릭/탭 | 나무 블록 또는 플라스틱 클릭 |
| 드래그 중 글자 추가 | 가벼운 틱(tick) | 연속적이고 리드미컬한 느낌 |
| 드래그 취소 | 부드러운 취소음 | 실망스럽지 않은 중성적 소리 |
| 잘못된 단어 | 부드러운 "bloop" | **절대 강한 버저 금지** |

### 10.3 레벨 클리어 팡파레

```
빌드업 (0.5초) → 클라이맥스 (1~2초) → 여운 (1초)
악기: 피아노 + 오케스트라 히트 + 종소리 + 파티클 효과음
패턴: 상승 멜로디 → 화음 해결 → 자연스럽게 BGM으로 연결
```

### 10.4 전체 효과음 사양표

| 효과음 | 길이 | 주파수 특성 | 비고 |
|--------|------|-----------|------|
| 타일 탭 | 0.05초 | 중고음 클릭 (1.5~3kHz) | 경쾌하되 크지 않게 |
| 단어 발견 (짧음) | 0.4초 | 상승 차임 C-E-G | 기분 좋은 성취감 |
| 단어 발견 (긺) | 0.8초 | 상승 차임 + 글리터 | 더 화려하게 |
| 보너스 단어 | 1.2초 | 팡파레 미니 | "WOW" 모먼트 |
| 레벨 클리어 | 2.0초 | 오케스트라 히트 | 달성감, 뿌듯함 |
| 오류/실패 | 0.2초 | 부드러운 "bloop" | **강한 버저 절대 금지** |
| 코인 획득 | 0.5초 | 딩딩 금속음 | 반짝이는 느낌 |
| 힌트 사용 | 0.6초 | 신비로운 글리터 | 마법 같은 느낌 |

### 10.5 실패/오류음 설계 철학

> ⚠️ **핵심 원칙**: 워드 퍼즐/힐링 게임에서 "실패음"은 최대한 부드럽고 중성적으로!
>
> - 강한 버저(BUZZ) 소리 = **절대 금지**. 사용자 불쾌감 → 이탈률 상승
> - 권장: 부드러운 "bloop" 또는 피아노 미스노트 1개
> - 선택적: 오류음 자체를 없애고 타일이 살짝 흔들리는 **시각 피드백만 제공**

```
테마별:   각 테마(animals, food, music 등)에 맞는 배경 팔레트
타일:     흰색 라운드 사각형. 선택 시 골드로 변경
분위기:   릴렉싱. 느긋한. 아름다운 자연/세계 배경
```

### 12.2 타일 이미지 제작 사양 (최종 권장)

```
형태:      라운드 사각형 (corner radius = 타일 크기의 20%)
기본 색상: #FFFFFF 또는 #FFFDE7 (따뜻한 흰색)
테두리:    1~2px, #E0E0E0 (또는 없음)
그림자:    drop shadow (0, 2px, 4px, rgba(0,0,0,0.15))
선택 색상: #FFD54F (따뜻한 골드)
정답 색상: #A5D6A7 (소프트 그린) → #4CAF50 (그린)
폰트:      Nunito Bold 또는 Fredoka One
글자 색상: #3E2723 (딥 브라운)
글자 크기: 타일 크기의 55~65%
```

### 12.3 BGM 제작 사양 (최종 권장)

```
장르:   앰비언트 + 자연 환경음 혼합
BPM:    70~85
키:     C Major 또는 G Major
구조:   레이어 4개 (앰비언트 / 멜로디 / 리듬 / 자연음)
루프:   90~120초
테마별: Layer 4 환경음만 교체하여 8가지 변형
```
